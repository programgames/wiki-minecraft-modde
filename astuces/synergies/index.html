    Synergies Cross-Mods¶

    Les combinaisons entre mods peuvent créer des setups extrêmement puissants. Ce guide présente les meilleures synergies.

    Synergies Tier S (Game-Breaking)¶

    Apotheosis + Silent Gear¶

    Combo Ultra-Puissant

    Résultat : Armes avec dégâts absurdes (1000+ par hit)

    1. Créer une arme Silent Gear avec les meilleurs matériaux (Netherite tip, Blaze Rod handle)
    2. Appliquer des enchantements Apotheosis niveau 100+ (Sharpness, Critical Strike)
    3. Insérer des Gems Ancient dans les sockets
    4. Utiliser la Reforging Table pour obtenir les meilleurs affixes

    AE2 + Create (via Mechanical Arm)¶

    Autocrafting Hybride

    Résultat : Create peut alimenter l'autocrafting AE2

    ME Interface → Chest → Mechanical Arm → Machine Create → Chest → ME Interface
    • Le Mechanical Arm de Create peut importer/exporter depuis une ME Interface
    • Permet d'intégrer les machines Create dans les patterns AE2
    • Utile pour : Mixing, Pressing, Cutting, etc.

    Occultism + Ars Nouveau (via Ars Ocultas)¶

    Magie Combinée

    Résultat : Invocations pilotées par sorts

    • Les esprits Occultism peuvent être invoqués avec des glyphes Ars
    • Automation magique sans limites de channels
    • Djinn Miner + Spell de téléportation = mining automatique n'importe où

    Botania + Apotheosis¶

    Mana Enchanting

    Résultat : Enchantements surpuissants via mana

    • Mana Enchanter de Botania + livres Apotheosis haut niveau
    • Permet de combiner des enchantements normalement incompatibles
    • Coût en mana très élevé mais résultat garanti

    Synergies Tier A (Très Puissantes)¶

    Mystical Agriculture + Nature's Aura¶

    Composant Effet
    Inferium Seeds Base de croissance
    Farming Stencil Zone de farming auto
    Lily Pad of Fertility Accélération x2-3
    Growth Accelerators (x64) Croissance quasi-instantanée

    Setup Optimal

    Placer 64 Growth Accelerators sous le Farming Stencil de Nature's Aura. Ajouter un Lily Pad of Fertility (Reliquary) à côté.

    Productive Bees + Mystical Agriculture¶

    • Certaines abeilles produisent directement de l'essence Mystical
    • Bee genes Very High productivity = production massive
    • Combiner avec Centrifuge automatisée vers AE2

    Mekanism + Industrial Foregoing¶

    Digital Miner (Mekanism) → Ores
          +
Laser Drill (IF) → Ressources rares
          =
Ressources infinies de TOUT

    Blood Magic + Spirit¶

    LP Infini

    Well of Suffering (Blood Magic) sur un Soul Cage (Spirit) = génération de LP passive infinie.

    Le Soul Cage spawn les mobs, Well of Suffering les damage et génère du LP.

    Flux Networks + Tout¶

    • Un seul Flux Controller peut alimenter toutes vos bases
    • Cross-dimension : Overworld, Nether, End, dimensions moddées
    • Plus besoin de câbles d'énergie nulle part

    Synergies Tier B (Utiles)¶

    Tinker's Construct + Apotheosis¶

    • Outils Tinkers avec enchantements Apotheosis niveau 100+
    • Moss modifier + Mending = réparation double
    • Attention : certains modpacks désactivent cette combo

    Create + Thermal (via Create Crafts & Additions)¶

    • Conversion RF ↔ Rotation bidirectionnelle
    • Alimenter des machines Thermal avec un Water Wheel
    • Ou alimenter Create avec un Dynamo

    Ars Nouveau + Productive Bees¶

    • Drygmy farms peuvent cibler les abeilles
    • Production de combs sans ruches
    • Combiner avec sorts AoE pour récolte massive

    PneumaticCraft + AE2¶

    • Drones PneumaticCraft peuvent interagir avec ME Interfaces
    • Automation conditionnelle impossible avec AE2 seul
    • Pressure Chamber pour enchantements spéciaux

    Synergies de Farming¶

    Combo Croissance Maximum¶

    Mod Item Bonus
    Mystical Agriculture Growth Accelerator x64 +6400%
    Reliquary Lily Pad of Fertility +200%
    Cyclic Sprinkler +100%
    Nature's Aura Farming Stencil Auto-harvest
    Thermal Phyto-Gro Instant grow

    Combo XP Maximum¶

    1. Mob Grinding Utils Mob Masher avec Looting
    2. Cyclic XP Pylon pour collecter
    3. Industrial Foregoing Mob Crusher pour Essence
    4. Apotheosis Spawner boosté au maximum

    Synergies de Combat¶

    Build "One-Shot Everything"¶

    Silent Gear Greatsword
  + Apotheosis Sharpness XV
  + Apotheosis Critical Strike V
  + Ancient Gem (Strength)
  + Curio Ring of Odin (Botania)
  + Strength Potion II
  = 2000+ dégâts par hit

    Build "Immortel"¶

    MekaSuit / Draconic Armor
  + Charm of Undying (Curio slot)
  + Ring of Last Chance (Nature's Aura)
  + Botania Totem of Undying infini
  + Blood Magic Living Armor (Self Sacrifice)
  = Quasi-impossible à tuer

    Anti-Synergies (À éviter)¶

    Conflits Connus

    • Refined Storage + AE2 : Choisir l'un ou l'autre, pas les deux
    • Multiple Ore Doublers : Peut causer des duplications non voulues
    • Chunk Loaders multiples : Cause du lag, en utiliser UN seul type
    • Apotheosis + certains mods d'enchant : Conflits de niveaux max

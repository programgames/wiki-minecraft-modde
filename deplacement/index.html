    Déplacement & Transport¶

    • Iron Jetpacks

      Jetpacks modulaires par tiers. Craftables avec n'importe quel métal, vol rapide et configurable.

      Découvrir

    • Waystones

      Téléportation entre waypoints nommés. Trouvez-les dans les villages ou craftez les vôtres.

      Découvrir

    • Explorer's Compass

      Localise n'importe quelle structure du jeu. Indispensable pour l'exploration efficace.

      Découvrir

    • Nature's Compass

      Trouve n'importe quel biome du jeu. Compatible avec tous les mods de biomes.

      Découvrir

    Comparaison Rapide¶

    Mod Difficulté Phase Recommandé pour
    Iron Jetpacks 2/5 Early → Late Vol accessible par tiers
    Waystones 1/5 Early → Late Téléportation de base
    Explorer's Compass 1/5 Early Trouver des structures
    Nature's Compass 1/5 Early Trouver des biomes
    Tempad 2/5 Mid → Late TP style Loki cross-dimension
    Carry On 1/5 Early → Late Porter coffres et machines
    Mekanism Jetpack 3/5 Mid → Late Vol rapide avec infrastructure
    Angel Ring 4/5 Late Vol créatif permanent

    Progression Recommandée¶

    Early Game

    • [RUSH] Waystones pour la téléportation de base (villages naturels)
    • [RUSH] Explorer's Compass + Nature's Compass pour trouver structures et biomes
    • [RUSH] Sleeping Bag pour dormir sans changer le spawn
    • [RUSH] Travelers Boots de Tinker's pour la vitesse de déplacement
    • [RUSH] Glider pour planer après une chute ou avec Slimesling

    Mid Game

    • Craft un Iron Jetpack tier basique pour le vol
    • Setup Tempad pour la téléportation style Loki
    • Upgrade vers des jetpacks de tier supérieur
    • Utiliser Carry On pour déplacer coffres et machines

    Late Game

    • Jetpack Nitro (Iron Jetpacks) ou Armored Jetpack (Mekanism)
    • Angel Ring pour le vol créatif permanent
    • Mekasuit avec module vol pour la meilleure armure volante
    • Flux Networks + Player Transmitter pour recharger en volant

    Synergies entre Mods¶

    Combinaisons Puissantes

    Slimesling + Glider
    Propulsion massive avec Slimesling puis planage avec le Glider = déplacement rapide gratuit
    Iron Jetpacks + Flux Networks
    Recharge automatique du jetpack via Player Transmitter
    Waystones + Explorer's Compass
    Trouvez les villages avec le compass, activez leurs waystones naturels
    Mekanism Jetpack + Gas Tank
    Emportez de l'Hydrogen supplémentaire pour les longs voyages
    Carry On + Waystones
    Téléportez-vous puis portez les machines vers votre nouvelle base

    Iron Jetpacks¶

    CurseForge | Wiki

    Quick Start

    1. Craft un Cell (batterie) avec le métal de ton choix
    2. Craft un Thruster avec le même métal
    3. Assemble le Jetpack sur la table de craft
    4. Charge le jetpack dans n'importe quel chargeur RF/FE
    5. Équipe dans le slot de plastron et appuie sur Space pour voler

    Système de Tiers

    • Chaque métal donne un tier différent de jetpack
    • Les tiers supérieurs = plus de vitesse, capacité et efficacité
    • Stone → Iron → Gold → Diamond → Emerald → Sapphire → Nitro
    • Le tier Nitro est le plus rapide du jeu

    Contrôles

    • Space : Monter
    • Shift : Descendre
    • H (configurable) : Toggle hover mode
    • Le hover mode maintient l'altitude automatiquement

    Gestion de l'Énergie

    • Toujours garder un chargeur portable ou flux network
    • Les jetpacks hauts tiers consomment beaucoup
    • Curios slot compatible pour libérer le plastron

    Progression Recommandée

    • Stone Jetpack : Premier vol accessible
    • Diamond Jetpack : Mid-game, très rapide
    • Nitro Jetpack : End-game, vitesse maximale
    Objet Priorité Description
    Stone Jetpack RUSH Premier jetpack craftable, très accessible
    Iron Jetpack Bas Upgrade bon rapport qualité/prix
    Diamond Jetpack Moyen Excellente vitesse mid-game
    Nitro Jetpack Top Jetpack le plus rapide, end-game
    Cell - Batterie, premier composant à crafter
    Thruster - Propulseur, second composant
    Capacitor - Améliore la capacité énergétique
    • Tier Comparison Video

    Waystones¶

    CurseForge | Wiki

    Quick Start

    1. Trouve un Waystone naturel dans un village (right-click pour activer)
    2. Nomme le waystone pour l'identifier facilement
    3. Craft un Warp Stone pour TP portable avec cooldown
    4. Craft des Waystones supplémentaires pour ta base
    5. Utilise Shareable Waystone pour permettre l'accès aux autres joueurs

    Types de Waystones

    • Waystone standard : Point fixe, personnel
    • Shareable Waystone : Accessible par tous les joueurs
    • Global Waystone : Visible par tous même sans activation préalable
    • Les waystones naturels dans les villages sont déjà activables

    Items de Téléportation

    • Warp Stone : TP vers n'importe quel waystone (cooldown)
    • Warp Scroll : TP unique consommable
    • Return Scroll : Retour au dernier waystone utilisé
    • Bound Scroll : Lié à un waystone spécifique (infini)

    Configuration Serveur

    • Le coût en XP peut être configuré (souvent gratuit en modpack)
    • Le cooldown du Warp Stone est configurable
    • Vérifier les settings du serveur avant de planifier

    Setup Optimal

    • Un Shareable Waystone à ta base principale
    • Des Waystones standard aux fermes et spots importants
    • Un Bound Scroll vers la base toujours dans l'inventaire
    • Un Warp Stone pour les urgences
    Objet Priorité Description
    Waystone RUSH Point de téléportation fixe, craft ou villages
    Warp Stone RUSH TP portable avec cooldown, indispensable
    Shareable Waystone Moyen Waystone accessible par tous les joueurs
    Bound Scroll Bas TP infini vers un waystone spécifique
    Return Scroll - Retour au dernier waystone utilisé
    Warp Scroll - TP unique consommable, craft de secours
    Portstone - Version portable du waystone
    • Waystones Tutorial Video

    Explorer's Compass¶

    CurseForge | Wiki

    Quick Start

    1. Craft l'Explorer's Compass (4 bois + 4 fer + 1 compass)
    2. Right-click pour ouvrir l'interface de sélection
    3. Choisis la structure que tu cherches dans la liste
    4. Suis la direction et distance affichées sur le HUD
    5. Répète pour chaque structure à trouver

    Utilisation

    • Affiche un HUD avec les coordonnées exactes
    • Cherche dans un rayon configurable (par défaut 5000 blocs)
    • Inclut les structures de tous les mods installés
    • Trie les structures par mod ou alphabétiquement

    Structures Importantes à Trouver

    • Village : Waystones naturels, trades, ressources
    • Stronghold : Portail de l'End
    • Mansion : Loot rare, Totems of Undying
    • Temples : Loot divers
    • Structures moddées : Twilight Forest, Blue Skies, etc.

    Limitations

    • La recherche peut prendre du temps sur les grandes distances
    • Certaines structures sont très rares (Mansion)
    • Le rayon de recherche impacte les performances

    Combo avec Waystones

    • Trouve un village avec Explorer's Compass
    • Active le Waystone naturel du village
    • Répète pour créer un réseau de téléportation
    Objet Priorité Description
    Explorer's Compass RUSH Localise n'importe quelle structure, indispensable
    • Explorer's Compass Guide

    Nature's Compass¶

    CurseForge | Wiki

    Quick Start

    1. Craft le Nature's Compass (4 saplings + 4 bois + 1 compass)
    2. Right-click pour ouvrir l'interface de sélection
    3. Choisis le biome que tu cherches dans la liste
    4. Suis la direction et distance affichées sur le HUD
    5. Compatible avec tous les biomes moddés (Biomes O' Plenty, etc.)

    Utilisation

    • Cherche les biomes de tous les mods installés
    • Affiche la distance et direction en temps réel
    • Le rayon de recherche est configurable
    • Affiche également les informations du biome actuel

    Biomes Importants

    • Mushroom Island : Pas de spawn de mobs
    • Ice Spikes : Packed Ice en abondance
    • Jungle : Temples, Pandas, Cocoa
    • Badlands : Or en surface, Terracotta
    • Biomes moddés : Ressources uniques selon les mods

    Biomes Rares

    • Certains biomes sont extrêmement rares
    • La recherche peut dépasser le rayon configuré
    • Les biomes moddés peuvent avoir des conditions spéciales

    Utilisation Combo

    • Trouve le biome avec Nature's Compass
    • Pose un Waystone sur place
    • Retourne facilement pour farmer les ressources
    Objet Priorité Description
    Nature's Compass RUSH Localise n'importe quel biome, indispensable
    • Nature's Compass Tutorial

    Mods Secondaires¶

    Tempad

    CurseForge

    Quick Start

    1. Craft le Tempad (matériaux end-game requis)
    2. Right-click pour ouvrir l'interface
    3. Sauvegarde des positions avec un nom personnalisé
    4. Sélectionne une destination et ouvre un portail orange
    5. Traverse le portail pour la téléportation instantanée

    Fonctionnalités

    • Téléportation style Loki avec portail visuel
    • Fonctionne cross-dimension (Overworld ↔ Nether ↔ End)
    • Le portail reste ouvert quelques secondes (traversable par d'autres)
    • Aucun coût en ressources, seulement un cooldown

    Organisation

    • Renomme les positions pour t'y retrouver
    • Catégorise par dimension ou type (fermes, bases, etc.)
    • Le portail peut être utilisé par d'autres joueurs à proximité

    Cooldown

    • Le Tempad a un cooldown entre utilisations
    • Planifie tes voyages en conséquence
    • Version He Who Remains n'a pas de cooldown (créatif)
    Objet Priorité Description
    Tempad Moyen Dispositif de téléportation principal
    He Who Remains' Tempad Top Version créatif sans cooldown
    Carry On

    CurseForge

    Quick Start

    1. Approche d'un bloc ou entité avec les mains vides
    2. Maintiens Shift + Right-click pour le soulever
    3. Tu portes maintenant le bloc avec son contenu
    4. Place-le avec Right-click à destination
    5. Fonctionne sur les coffres, machines, animaux et villageois

    Blocs Transportables

    • Coffres avec tout leur contenu
    • Machines avec leur état (progress, inventory)
    • Spawners (si autorisé par le serveur)
    • Ruches avec leurs abeilles

    Entités Transportables

    • Animaux de ferme
    • Villageois (très utile!)
    • Certains mobs passifs

    Limitations

    • Mouvement ralenti pendant le transport
    • Impossible d'interagir avec d'autres choses
    • Certains blocs sont trop lourds (obsidian, etc.)
    • Configuration serveur peut interdire certains blocs
    Objet Priorité Description
    Aucun item requis RUSH Fonctionne avec les mains vides uniquement
    Mekanism Jetpack

    CurseForge | Wiki

    Quick Start

    1. Setup une production d'Hydrogen (Electrolytic Separator + Eau)
    2. Craft le Jetpack de Mekanism
    3. Remplis-le d'Hydrogen via un Chemical Tank ou Pressurized Tube
    4. Équipe dans le slot de plastron
    5. Appuie sur Space pour voler

    Types de Jetpacks Mekanism

    • Jetpack standard : Vol avec Hydrogen
    • Armored Jetpack : Protection + vol
    • Mekasuit Jetpack Module : Intégré à l'armure end-game

    Gestion de l'Hydrogen

    • Emporte un Portable Chemical Tank plein
    • Setup un Personal Gas Tank pour recharger
    • L'Hydrogen est facile à produire (eau + électricité)

    Consommation

    • Consomme de l'Hydrogen en permanence pendant le vol
    • Plus rapide que Iron Jetpacks mais nécessite infrastructure
    • Le Mekasuit est l'upgrade ultime
    Objet Priorité Description
    Jetpack Moyen Vol avec Hydrogen, rapide
    Armored Jetpack Haut Jetpack + protection
    Mekasuit + Module Top Meilleur jetpack end-game
    Portable Chemical Tank Bas Réserve d'Hydrogen portable
    Angel Ring

    CurseForge

    Quick Start

    1. Accumule les ressources pour le craft (très coûteux)
    2. Craft l'Angel Ring selon la recette du modpack
    3. Équipe dans un slot Curios (ou baubles)
    4. Active le vol créatif avec double Space ou keybind
    5. Vol infini sans consommation d'énergie

    Avantages

    • Vol créatif permanent
    • Aucune consommation d'énergie ou fuel
    • Compatible avec le slot Curios (garde ton plastron)
    • Vol stationnaire parfait pour le building

    Craft

    • Le craft varie selon le modpack
    • Généralement très coûteux (end-game)
    • Peut nécessiter des Nether Stars ou items rares

    Late Game Only

    • Le coût de craft est prohibitif en early
    • Les jetpacks sont plus accessibles avant
    • À viser comme objectif end-game
    Objet Priorité Description
    Angel Ring Top Vol créatif permanent, objectif ultime
    Pretty Pipes

    CurseForge

    Quick Start

    1. Craft des Pipes avec les lingots appropriés
    2. Connecte les inventaires source et destination
    3. Ajoute un Extraction Module côté source
    4. Ajoute des Filters pour contrôler ce qui passe
    5. Upgrade avec des modules de vitesse et stack size

    Système de Modules

    • Extraction Module : Tire les items de l'inventaire
    • Filter Module : Contrôle ce qui passe
    • Speed Module : Accélère le transfert
    • Stack Size Module : Transfère plus d'items à la fois

    Crafting et Requêtes

    • Crafting Module : Auto-craft sur demande
    • Retrieval Module : Demande des items spécifiques
    • Système de requêtes comme AE2/RS mais plus simple

    Comparaison

    • Plus simple que AE2/RS pour les débutants
    • Moins puissant pour les systèmes complexes
    • Excellent pour les premiers automations
    Objet Priorité Description
    Pipe Bas Connecteur de base
    Extraction Module Bas Nécessaire pour tirer les items
    Filter Module - Contrôle du flux
    Speed Module - Accélération du transfert
    Item Terminal Bas Interface de requête centralisée
    Elevators

    CurseForge

    Quick Start

    1. Craft deux Elevator Blocks de la même couleur
    2. Place un bloc au niveau bas
    3. Place l'autre directement au-dessus au niveau haut
    4. Monte sur l'ascenseur et saute pour monter
    5. Sneak pour descendre au niveau inférieur

    Configuration

    • Les blocs doivent être de la même couleur
    • Peuvent être séparés par n'importe quelle distance verticale
    • Supporte plusieurs niveaux (bas → mid → haut)

    Utilisation

    • Parfait pour les bases multi-étages
    • Fonctionne avec la redstone pour automation
    • Compatible avec Carry On pour transport

    Limitations

    • Alignement vertical strict requis
    • Ne fonctionne pas en diagonale
    • Distance max configurable par serveur
    Objet Priorité Description
    Elevator Block Bas Téléportation verticale instantanée
    Colored Variants - 16 couleurs disponibles
    Elytra Slot / Curious Elytra

    Elytra Slot | Curious Elytra

    Quick Start

    1. Installe le mod compatible avec ton modpack
    2. L'Elytra peut maintenant être équipée en slot Curios
    3. Tu peux porter Elytra + Plastron en même temps!
    4. Active le vol avec le double-jump
    5. Combine avec un jetpack pour le décollage

    Avantages Majeurs

    • Elytra + Armure en même temps = plus de protection
    • Compatible avec les Elytra enchantées
    • Fonctionne avec les skins d'Elytra moddées
    • Garde ton slot de plastron pour armure ou jetpack

    Combinaisons

    • Elytra + Jetpack : Décollage vertical + plané
    • Elytra + Mekasuit : Meilleur des deux mondes
    • Utilise des Fireworks pour booster le vol
    Objet Priorité Description
    Elytra (slot Curios) - Vol + protection
    Firework Rockets - Boost de vitesse
    Paragliders

    CurseForge

    Quick Start

    1. Craft un Paraglider (Deku Leaf ou Paraglider)
    2. Équipe dans ta main ou slot Curios
    3. Saute et active pour planer
    4. Utilise la stamina pour voler
    5. Récupère de la stamina au sol

    Système de Stamina

    • Inspiré de Breath of the Wild
    • La stamina se recharge au sol
    • Plus tu voles, plus tu consommes
    • Des items augmentent la stamina max

    Types de Gliders

    • Deku Leaf : Version early game
    • Paraglider : Version améliorée
    • Différentes couleurs disponibles

    Limitations

    • Pas de vol vertical (seulement plané)
    • Nécessite un point haut pour décoller
    • Combine avec Slimesling pour la propulsion
    Objet Priorité Description
    Deku Leaf RUSH Planeur early game
    Paraglider - Planeur amélioré
    Spirit Orbs Augmentent la stamina
    Stamina Vessels Plus de stamina
    Slimesling + Glider Combo

    Quick Start

    1. Craft un Slimesling (Tinker's Construct)
    2. Craft un Hang Glider (OpenBlocks ou autre)
    3. Utilise le Slimesling pour te propulser en l'air
    4. Déploie immédiatement le Glider en l'air
    5. Plane sur de longues distances gratuitement

    Technique

    • Charge le Slimesling en maintenant right-click
    • Plus tu charges, plus tu vas loin et haut
    • Déploie le Glider au sommet de la trajectoire
    • Avec la bonne technique, traverse des centaines de blocs

    Avantages

    • Aucune énergie requise
    • Accessible très early game
    • Fun et satisfaisant à maîtriser

    Inconvénients

    • Demande de la pratique
    • Pas de vol vertical (seulement propulsion + plané)
    • Moins pratique que les jetpacks
    Objet Priorité Description
    Slimesling RUSH Propulsion puissante, Tinker's Construct
    Hang Glider RUSH Planage, divers mods

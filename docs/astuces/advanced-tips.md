# Astuces Avancées

Techniques de pro pour optimiser votre gameplay.

## Mouse Tweaks & Inventory Management

### Raccourcis Souris (Mouse Tweaks)

!!! success "Gestion d'Inventaire Ultra-Rapide"

| Action | Raccourci | Effet |
|--------|-----------|-------|
| Drag gauche | Maintenir + glisser | Distribue items également |
| Drag droit | Maintenir + glisser | Place 1 item par slot |
| Shift + Drag | Shift + glisser | Transfert rapide multiple |
| Scroll sur stack | Molette | Transfert 1 item à la fois |
| Shift + Scroll | Shift + molette | Transfert stack entier |
| Double-clic | 2x clic gauche | Ramasse tous les items identiques |

### Crafting Tweaks

!!! tip "Mod indispensable"
    | Raccourci | Effet |
    |-----------|-------|
    | `R` | Rotation du craft |
    | `B` | Équilibre les items dans la grille |
    | `C` | Clear la grille vers l'inventaire |
    | Shift + Clic résultat | Craft maximum possible |

### Tips Généraux Inventaire

- **F3 + H** : Active les tooltips avancés (durabilité, NBT, ID)
- **Shift + Clic** : Transfert rapide
- **Nombres 1-9** : Swap avec la hotbar
- **Q** : Drop 1 item
- **Ctrl + Q** : Drop stack entier
- **Middle Click (créatif)** : Clone l'item

### Contourner les Crafts Coûteux

!!! example "Exemples"

    **Angel Ring trop cher?**
    → Chercher dans les **Loot Crates** des donjons
    → Certains villagers le tradent
    → Quêtes qui le donnent en récompense

    **Nether Star sans Wither?**
    → **HNN** (Hostile Neural Networks) simulation
    → Certaines **bees** produisent des Nether Stars
    → **Drygmy** farm sur un Wither contenu

- Les **Affixes** sur les mobs = meilleur loot

## Villager Trading Optimization

!!! success "Meilleurs Trades du Jeu"

### Setup Optimal

```
1. Capturer un Villager (bateau ou minecart)
2. Infecter avec Zombie (difficulté Hard)
3. Soigner avec Weakness + Golden Apple
4. Répéter 5x pour prix minimum
5. Placer le workstation voulu
```

### Meilleurs Trades par Profession

| Profession | Workstation | Trade Intéressant |
|------------|-------------|-------------------|
| Librarian | Lectern | TOUS les enchantements |
| Armorer | Blast Furnace | Diamond armor |
| Toolsmith | Smithing Table | Diamond tools |
| Weaponsmith | Grindstone | Diamond sword/axe |
| Cleric | Brewing Stand | Ender Pearls, Redstone |
| Farmer | Composter | Golden Carrots, Emeralds |
| Cartographer | Cartography | Explorer Maps |
| Fletcher | Fletching Table | Arrows, Crossbows |

### Librarian Cycling

!!! tip "Obtenir N'importe Quel Enchantement"
    1. Placer/retirer le **Lectern** jusqu'au bon trade
    2. Trader une fois pour **lock** les trades
    3. Répéter pour chaque enchantement voulu
    4. Max **Mending** et **Silk Touch** en priorité!

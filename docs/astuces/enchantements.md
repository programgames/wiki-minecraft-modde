## Meilleurs Enchantements par Type

### Épées

| Enchantement | Niveau Max | Utilisation | Recommandation |
|--------------|------------|-------------|----------------|
| **Sharpness** | V (X avec Apotheosis) | Dégâts universels | Recommandé pour usage général |
| **Smite** | V (X avec Apotheosis) | +dégâts vs morts-vivants | Meilleur DPS contre Wither/Zombies |
| **Bane of Arthropods** | V | +dégâts vs araignées | Situationnel |
| **Looting** | III (VII avec Apotheosis) | +drops de mobs | **Indispensable** |
| **Sweeping Edge** | III | +dégâts de balayage | Excellent pour farming |
| **Fire Aspect** | II | Dégâts de feu | Attention: peut détruire certains drops |
| **Knockback** | II | Repousse les mobs | Éviter pour le combat rapproché |

**Verdict Sharpness vs Smite:**
- **Sharpness** : Polyvalent, fonctionne sur tous les mobs
- **Smite** : +2.5 dégâts/niveau vs morts-vivants (meilleur pour Wither boss)
- Conseil : Avoir une épée de chaque pour les situations

### Armure

| Enchantement | Niveau Max | Effet | Recommandation |
|--------------|------------|-------|----------------|
| **Protection** | IV (VII avec Apotheosis) | Réduction tous dégâts | **Best all-around** |
| **Fire Protection** | IV | Réduction feu | Situationnel (Nether) |
| **Blast Protection** | IV | Réduction explosions | Situationnel (Creepers) |
| **Projectile Protection** | IV | Réduction projectiles | Situationnel (Skeletons) |
| **Thorns** | III | Renvoie dégâts | **Controversé** |
| **Unbreaking** | III | Durabilité | Toujours |
| **Mending** | I | Réparation | **Indispensable** |

**Le Débat Thorns:**

| Avantages | Inconvénients |
|-----------|---------------|
| Dégâts passifs aux attaquants | Consomme durabilité rapidement |
| Utile en early game | Inutile en late game (one-shot mobs) |
| Bon pour AFK farms | Peut tuer accidentellement des mobs utiles |

**Verdict:** Éviter Thorns en late game, la durabilité est plus importante.

## Comment Enchanter Efficacement

### Comparaison des Méthodes

| Méthode | Niveau Max | Coût XP | Contrôle |
|---------|------------|---------|----------|
| Table Vanilla | 30 | Moyen | Faible |
| Apotheosis Table | 100+ | Élevé | Total |
| Villageois | Variable | Émeraudes | Moyen |
| Loot Chests | Variable | Aucun | Aucun |
| Anvil Combining | N/A | Très élevé | Total |

---

### Vanilla Enchanting Table Setup

```
Configuration optimale (15 Bookshelves):

    B B B B B
    B       B
    B   T   B
    B       B
    B B B B B

T = Table d'enchantement
B = Bibliothèque (Bookshelf)
```

- Minimum 15 bibliothèques pour niveau 30
- Espace d'un bloc entre table et bibliothèques
- Torches au sol OK, pas sur les murs

---

### Stratégies de Combinaison à l'Enclume

**Règle d'or: Toujours combiner items de même "work penalty"**

| Combinaisons | Work Penalty |
|--------------|--------------|
| Item neuf | 0 |
| 1ère combinaison | 1 |
| 2ème combinaison | 2 |
| etc. | +1 par opération |

**Ordre optimal pour enchanter un item:**

1. Enchanter des livres individuels
2. Combiner livres par paires
3. Combiner les paires
4. Appliquer le super-livre à l'item

```
Exemple pour épée parfaite:

Livre A: Sharpness V      Livre B: Looting III
        ↘                ↙
         Livre AB (combiné)

Livre C: Unbreaking III   Livre D: Mending
        ↘                ↙
         Livre CD (combiné)

         Livre AB    +    Livre CD
                ↘    ↙
              Livre ABCD
                   ↓
              Épée Parfaite
```

## Apotheosis Enchanting

### Le Système Apotheosis

Apotheosis révolutionne l'enchantement avec trois statistiques:

| Statistique | Effet | Source |
|-------------|-------|--------|
| **Eterna** | Niveau max d'enchantement | Bookshelves, candles |
| **Quanta** | Variance des enchantements | Hellshelves, specific items |
| **Arcana** | Chance de treasure enchants | Seashells, specific items |

---

### Configuration des Étagères

**Blocs et leurs stats:**

| Bloc | Eterna | Quanta | Arcana |
|------|--------|--------|--------|
| Bookshelf | +1 | 0 | 0 |
| Hellshelf | +1.5 | +5% | 0 |
| Seashell Block | +1.25 | 0 | +5% |
| Infused Seashelf | +2.5 | 0 | +15% |
| Melonshelf | +1 | 0 | +10% |
| Beeshelf | +1 | +2.5% | +2.5% |
| Soul-Touched Hellshelf | +1.5 | +10% | 0 |
| Blazing Hellshelf | +2 | +20% | 0 |
| Endshelf | +2 | -5% | +10% |
| Draconic Endshelf | +5 | 0 | +25% |

---

### Setup pour Max Level Enchanting

**Objectif: 100 Eterna, 0-100% Quanta, haute Arcana**

```
Configuration recommandée:

                Candle Row
    ES  ES  ES  DE  ES  ES  ES
    ES              ES
    ES      TABLE       ES
    ES              ES
    ES  ES  ES  DE  ES  ES  ES

ES = Endshelf
DE = Draconic Endshelf
+ Candles pour Eterna supplémentaire
```

**Étapes pour max setup:**

1. Placer la table d'enchantement
2. Entourer de Endshelves (2 blocs de distance)
3. Ajouter Draconic Endshelves pour Arcana
4. Utiliser des bougies pour atteindre 100 Eterna
5. Ajuster Quanta selon besoin

### Treasure Enchants

Les **Treasure Enchants** sont des enchantements rares qui n'apparaissent pas normalement:

| Enchantement | Effet | Source |
|--------------|-------|--------|
| **Spearfishing** | Attrape mobs aquatiques | Arcana élevée |
| **Chainsaw** | Coupe arbres entiers | Arcana élevée |
| **Nature's Blessing** | Croissance des plantes | Arcana élevée |
| **Reflecting** | Renvoie les projectiles | Arcana élevée |
| **Inert Gravity** | Pas de fall damage | Arcana élevée |

**Pour obtenir des Treasure Enchants:**
- Arcana minimum de 25%
- Plus l'Arcana est haute, meilleure est la chance
- Certains nécessitent 75%+ Arcana

## Automatisation de l'Enchantement

### Setup Auto-Enchant

> **Référence visuelle:** [Guide Auto-Enchant ATM](https://imgur.com/gallery/how-to-autoenchant-atm6-sky-FSIsYE6)

**Composants nécessaires:**

| Bloc | Fonction |
|------|----------|
| Enchanting Table | Enchante les items |
| Experience Obelisk | Stocke l'XP |
| Hopper/Pipe | Transport items |
| Chest | Input/Output |
| Redstone | Automatisation |

**Configuration de base:**

```
[Chest Input]
      ↓
[Hopper/Pipe]
      ↓
[Enchanting Table]
      ↓
[Hopper/Pipe]
      ↓
[Chest Output]

+ Experience Obelisk adjacent à la table
+ Redstone pour activer/désactiver
```

**Mods utiles pour l'automatisation:**

| Mod | Bloc | Fonction |
|-----|------|----------|
| Modular Routers | Item Router | Placement d'items |
| Integrated Dynamics | Logic | Filtrage intelligent |
| Create | Deployer | Simule clic joueur |
| Cyclic | Mechanical User | Simule clic joueur |

## Early Game (Jour 1-3)

### Slimesling + Glider (Gratuit)
!!! tip "Option Zéro Coût"
    **Mods** : Tinker's Construct + OpenBlocks

    1. Craft un **Slimesling** (slime balls)
    2. Craft un **Glider** (leather + sticks)
    3. Utilise le Slimesling pour te propulser
    4. Ouvre le Glider en l'air (clic droit ou keybind)

    **Avantages** : Totalement gratuit, disponible très tôt
    **Inconvénients** : Courbe d'apprentissage, pas de hover

### Brooms (EvilCraft / Hexerei)
!!! success "Vol Early Accessible"
    **Craft EvilCraft** : Dark Sticks + Broom Parts

    **Craft Hexerei** : Broom + upgrades

    - Vol contrôlé dès le début
    - Pas de consommation d'énergie
    - Vitesse modérée mais suffisante

### Hang Glider (OpenBlocks)
!!! note "Glide Seulement"
    - Ne permet pas de monter, seulement de planer
    - Parfait combiné avec Slimesling ou hauteur naturelle
    - Craft très simple (leather + sticks)

## Mid Game

### Iron Jetpacks (Tiered)
!!! example "Évolutif"
    | Tier | Vitesse | RF Storage | Craft |
    |------|---------|------------|-------|
    | Stone | Lent | 20,000 | Très facile |
    | Iron | Moyen | 80,000 | Facile |
    | Gold | Rapide | 200,000 | Moyen |
    | Diamond | Très rapide | 800,000 | Coûteux |
    | Emerald | Ultra rapide | 2,000,000 | Très coûteux |
    | Netherite | Maximum | 10,000,000 | End-game |

    **Tips** :
    - Commencer par **Iron** pour un bon ratio coût/perf
    - Les jetpacks peuvent être **enchantés** (Unbreaking)
    - Keybind pour hover mode!

### Mekanism Jetpack
!!! tip "Intégration Mekanism"
    - Fonctionne au **Hydrogen** (gratuit avec eau + énergie)
    - S'intègre plus tard dans le **MekaSuit**
    - Hover mode inclus

    **Craft** : Steel + Gas Tank + modules

### Create Jetpack (Addon)
!!! note "Style Create"
    - Fonctionne avec la **rotation** Create
    - Backtank requis pour le carburant
    - Esthétique steampunk

### Angel Ring (Extra Utilities)
!!! danger "Creative Flight Mid-Game"
    **Le graal du mid-game** - Vol créatif sans consommation!

    **Craft** (varie selon modpack) :
    - Souvent nécessite des matériaux de boss
    - Nether Star, Diamond Block, etc.
    - Vérifier dans JEI

    **Variantes** :
    - Différentes couleurs = différents crafts
    - Certaines ont des effets spéciaux

### Elytra + Firework Rockets
!!! warning "Rapide mais Consommable"
    - **Elytra** : Loot End Ships ou craft (certains mods)
    - **Firework Rockets** : Gunpowder + Paper

    **Tips** :
    - Enchanter avec **Unbreaking III** et **Mending**
    - Rocket Duration 3 = plus de boost
    - Peut être combiné avec autres moyens de vol

## Late Game

### Flügel Tiara (Botania)
!!! success "Vol Créatif Magique"
    - **Creative flight** alimenté par Mana
    - Craft avec Terrasteel et items Botania end-game
    - Très fluide et agréable

    **Coût** : Environ 1 mana par tick en vol

### Draconic Evolution Chestplate
!!! danger "Le Meilleur Vol du Jeu"
    - **Vol créatif** intégré
    - **Vitesse ajustable** (shift + scroll)
    - Peut atteindre des vitesses **absurdes**
    - RF powered

    **Modules** :
    - Flight module (obligatoire)
    - Speed module (recommandé)
    - Shield module (survie)

### MekaSuit Chestplate
!!! tip "Modulaire"
    - **Jetpack Module** : Vol standard
    - **Gravitational Modulating Module** : Creative flight
    - Combiner avec autres modules (shield, vision)

    **Avantage** : Une seule armure pour tout

## Tips Avancés

!!! success "Conseils Pro"

    1. **Toujours un backup** - Avoir un Elytra dans l'inventaire au cas où
    2. **Hover keybind** - Configurer une touche pour hover (évite les crashs)
    3. **Feather Falling** - Sur les bottes, au cas où
    4. **Charged items** - Toujours partir avec énergie pleine
    5. **Curio slot** - Certains items de vol vont en Curio (libère le chest slot)

!!! warning "Attention aux Dimensions"
    - Certains mods **désactivent le vol** dans certaines dimensions
    - Vérifier avant d'entrer dans une dimension inconnue
    - Twilight Forest, certains donjons, etc.

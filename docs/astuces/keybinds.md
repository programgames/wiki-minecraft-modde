# Raccourcis Clavier Complets

!!! keyboard "Guide des Touches"
    Cette page référence **tous les raccourcis clavier** essentiels pour jouer efficacement sur le serveur. Apprenez-les pour gagner un temps précieux !

---

## Raccourcis Essentiels

!!! tip "Les 10 touches à connaître absolument"
    Ces raccourcis sont utilisés **constamment** - apprenez-les en priorité !

| Action | Touche | Mod | Notes |
|--------|--------|-----|-------|
| **Ultimine** (minage en masse) | ++grave++ ou ++tilde++ | FTB Ultimine | Maintenir + casser un bloc |
| **Voir recette** | ++r++ | JEI | Sur un item dans l'inventaire |
| **Voir utilisations** | ++u++ | JEI | Où l'item est utilisé |
| **Trier inventaire** | ++middle-button++ | Inventory Tweaks | Clic molette sur l'inventaire |
| **Recherche inventaires** | ++y++ | Inventory Essentials | Cherche dans coffres proches |
| **Toggle Magnet** | ++m++ | Magnet Mod | Active/désactive l'aimant |
| **Curios GUI** | ++g++ | Curios | Équipement spécial |
| **Waystones** | ++b++ | Waystones | Menu téléportation |
| **Vein Mine** | ++grave++ | Vein Mining | Miner toute la veine |
| **Jetpack** | ++space++ (x2) | Iron Jetpacks | Double saut pour activer |

## Par Catégorie

=== "JEI / Recherche"

    !!! abstract "Navigation dans JEI"
        JEI (Just Enough Items) est votre **encyclopédie de craft** - maîtrisez ces touches !

    | Action | Touche par défaut | Mod | Notes |
    |--------|-------------------|-----|-------|
    | Voir la recette | ++r++ | JEI | Hover + appuyer |
    | Voir les utilisations | ++u++ | JEI | Où l'item est craft |
    | Ajouter aux favoris | ++a++ | JEI | Clic droit pour retirer |
    | Afficher favoris | ++a++ (maintenir) | JEI | Liste personnelle |
    | Rechercher | ++ctrl+f++ | JEI | Focus barre de recherche |
    | Page suivante | ++page-down++ | JEI | Liste d'items |
    | Page précédente | ++page-up++ | JEI | Liste d'items |
    | Mode triche | ++ctrl++ + clic | JEI | Obtenir l'item (créatif) |
    | Cacher un item | ++ctrl+h++ | JEI | Masquer de la liste |
    | Montrer cachés | ++ctrl+h++ | JEI | Toggle affichage |
    | Recherche par mod | `@modname` | JEI | Ex: `@mekanism` |
    | Recherche par tooltip | `#tooltip` | JEI | Ex: `#fire` |
    | Recherche par tag | `$tag` | JEI | Ex: `$forge:ingots` |

    !!! example "Syntaxe de recherche avancée"
        ```
        @mekanism steel    → Items du mod Mekanism contenant "steel"
        #upgrade           → Items avec "upgrade" dans le tooltip
        $forge:ingots      → Tous les lingots (tag Forge)
        ^advanced          → Items commençant par "advanced"
        ```

=== "Mining / Outils"

    !!! pickaxe "Minage Efficace"
        Ces raccourcis transforment le minage en expérience agréable.

    | Action | Touche par défaut | Mod | Notes |
    |--------|-------------------|-----|-------|
    | **FTB Ultimine** | ++grave++ ou ++tilde++ | FTB Ultimine | Maintenir pendant minage |
    | Mode Ultimine | ++grave+m++ | FTB Ultimine | Cycle des formes |
    | Vein Mining | ++grave++ | Vein Mining | Miner veine entière |
    | Tool cycling | ++v++ | Tool Belt | Changer d'outil rapide |
    | Excavate mode | ++shift+grave++ | Ultimine | Mode excavation |
    | AOE Toggle | ++shift+scroll++ | Certains outils | Active/désactive AOE |
    | Hammer mode | ++h++ | Silent Gear | Si configuré |
    | Builder's Wand | ++r++ (sur wand) | Building Gadgets | Mode construction |
    | Gadget GUI | ++g++ (sur gadget) | Building Gadgets | Configuration |
    | Copy/Paste | ++ctrl+c++ / ++ctrl+v++ | Building Gadgets | Copier structures |

    !!! warning "FTB Ultimine - Formes disponibles"
        Appuyez sur ++grave+m++ pour cycler entre :

        - **Shapeless** : Tous les blocs connectés du même type
        - **Small Tunnel** : Tunnel 3x3
        - **Large Tunnel** : Tunnel 5x5
        - **Mining Tunnel** : Tunnel 1x3

## Configurer ses Touches

### Accéder aux Options

!!! settings "Menu des contrôles"
    1. ++escape++ → **Options**
    2. **Contrôles** → **Touches**
    3. Rechercher par nom ou mod
    4. Cliquer sur la touche → appuyer sur nouvelle touche

### Conseils de Configuration

!!! tip "Recommandations"

    === "Souris Gaming"
        Si vous avez des boutons supplémentaires sur votre souris :

        | Bouton | Action recommandée |
        |--------|-------------------|
        | Bouton 4 (pouce) | FTB Ultimine |
        | Bouton 5 (pouce) | Toggle Magnet |
        | Molette clic | Trier inventaire |
        | Molette gauche | Zoom |
        | Molette droite | Waypoints |

    === "Clavier Standard"
        Organisation pour clavier standard :

        | Zone | Touches | Actions |
        |------|---------|---------|
        | Main gauche | ++q++ ++e++ ++r++ ++f++ | Actions fréquentes |
        | Pouce | ++space++ ++alt++ | Saut, actions secondaires |
        | Auriculaire | ++shift++ ++ctrl++ | Modifieurs |
        | Au-dessus de Tab | ++grave++ ++1++ - ++5++ | Ultimine, hotbar |

## Conflits à Éviter

!!! danger "Conflits Fréquents"
    Ces combinaisons causent souvent des problèmes :

| Conflit | Touches concernées | Solution |
|---------|-------------------|----------|
| JEI vs Chat | ++r++ pour recette vs reply | Changer reply en ++shift+r++ |
| Curios vs Gadgets | ++g++ pour les deux | Changer Gadgets en ++shift+g++ |
| Magnet vs Map | ++m++ pour les deux | Changer Map en ++n++ |
| Zoom vs Crawl | ++c++ pour les deux | Changer crawl en ++z++ |
| Waystones vs JourneyMap | ++b++ pour les deux | Changer l'un en ++backslash++ |

!!! warning "Vérifier les conflits"
    Dans le menu des touches, les **conflits apparaissent en rouge**. Réglez-les pour éviter des comportements inattendus !

## Résumé

!!! success "Points clés"
    - **Apprenez les 10 touches essentielles** en haut de page
    - **Résolvez les conflits** (affichés en rouge dans les options)
    - **Personnalisez** selon votre matériel (souris gaming = avantage)
    - **Pratiquez** - les raccourcis deviennent naturels avec le temps

!!! tip "Conseil final"
    Créez une **sauvegarde de vos options** (`options.txt` dans `.minecraft`) après avoir configuré vos touches. Vous pourrez les restaurer après une réinstallation !

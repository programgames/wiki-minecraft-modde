## Progression Recommandée

!!! tip "Early Game"
    - :zap: **[RUSH]** **Stirling Generator** (Mekanism) ou **Coal Generator** (tout mod tech)
    - :zap: **[RUSH]** **Steam Dynamo** de Thermal (eau + charbon)
    - :zap: **[RUSH]** **Furnator** de Powah (brûle des items)
    - :zap: **[RUSH]** **Heat Generator** de Mekanism (lave passive)
    - Chercher `generator` et `solar` dans JEI
    - Explorer les générateurs solaires basiques

!!! note "Mid Game"
    - **Magmatic Dynamo** ou **Magmator** avec lave du Nether
    - **Solar Panels** pour énergie gratuite
    - Setup **Flux Networks** pour distribution centralisée
    - **Compression Dynamo** avec fuel liquide
    - Transition vers les **Reactors** de Powah

!!! success "Late Game"
    - **Fusion Reactor** de Mekanism (millions de RF/t)
    - **Draconic Reactor** pour énergie massive (attention explosions)
    - **Extreme Reactors** optimisés
    - **Powah Nitro Reactor** pour production stable
    - Distribution globale via **Flux Networks**

## Synergies entre Mods

!!! abstract "Combinaisons Puissantes"

    **Flux Networks + Tout générateur**
    :   Distribution sans fil cross-dimension = une seule source pour toutes les bases

    **Mekanism + Thermal**
    :   Ethylene Generator avec biomasse + Dynamos pour backup

    **Powah Ender Cells + Machines distantes**
    :   Transfert sans fil simple sans setup Flux Networks

    **Thermal Augments + Dynamos**
    :   Transmission Coils stackés = output massif par dynamo

    **Mekanism Fission + Turbine**
    :   Chaleur → Steam → Turbine pour maximiser le rendement

## Powah

[CurseForge](https://www.curseforge.com/minecraft/mc-mods/powah-rearchitected) | [Wiki](https://ftb.fandom.com/wiki/Powah)

!!! success "Quick Start"
    1. Craft un **Furnator** basique pour commencer
    2. Upgrade vers **Magmator** avec lave du Nether
    3. Craft un **Energizing Orb** pour les crafts spéciaux (Uraninite)
    4. Connecte avec des **Energy Cables** de ton tier
    5. Évolue vers les **Reactors** pour production massive

=== "Tips"

    !!! tip "Système de Tiers"
        - Tous les générateurs ont **7 tiers** : Starter → Basic → Hardened → Blazing → Niotic → Spirited → Nitro
        - Chaque tier augmente **production ET capacité**
        - L'**Energizing Orb** est requis pour fabriquer les matériaux avancés

    !!! tip "Transfert d'Énergie"
        - **Ender Cells** transfèrent l'énergie **sans fil et cross-dimension**
        - **Ender Gates** connectent les machines au réseau Ender Cell
        - **Player Transmitter** charge les items dans l'inventaire du joueur

    !!! warning "Thermo Generator"
        - Placer entre une source **chaude** (lave, magma) et **froide** (eau, glace)
        - Plus la différence de température est grande, plus la production est élevée
        - La **Blazing Crystal** augmente la chaleur côté chaud

    !!! example "Setup Optimal Reactor"
        - Les **Reactors** nécessitent de l'**Uraninite** (craft via Energizing Orb)
        - Entourer de **Reactor Blocks** pour augmenter l'efficacité
        - Tier Nitro = production end-game stable

=== "Objets Importants"

    | Objet | Priorité | Description |
    |-------|:--------:|-------------|
    | Energizing Orb | :zap: RUSH | Crafting avec énergie (obligatoire pour progression) |
    | Furnator | :zap: RUSH | Génère avec combustibles solides |
    | Magmator | :star: | Génère avec lave (efficace mid-game) |
    | Thermo Generator | :star: | Génère avec différence de température |

    | Ender Gate | :star: | Connecte au réseau Ender Cell |
    | Energy Cable | :star: | Câbles d'énergie (tiers) |

## Mekanism Generators

[CurseForge](https://www.curseforge.com/minecraft/mc-mods/mekanism-generators) | [Wiki Officiel](https://wiki.aidancbrady.com/wiki/Generators)

!!! success "Quick Start"
    1. Craft un **Heat Generator** (génération passive avec lave)
    2. Setup un **Gas-Burning Generator** avec Ethylene (bio-fuel)
    3. Explore les **Solar/Wind Generators** pour énergie passive
    4. Progresse vers le **Fission Reactor** (attention au meltdown!)
    5. End-game : **Fusion Reactor** pour millions de RF/t

=== "Tips"

    !!! tip "Générateurs Simples"
        - **Heat Generator** : Place de la lave en dessous pour génération passive
        - **Solar Generator** : Upgrade vers **Advanced Solar** pour plus de production
        - **Wind Generator** : Plus haut = plus de production (Y>80 idéal)
        - **Bio Generator** : Utilise Bio Fuel (crushé depuis plantes)

    !!! tip "Ethylene Setup"
        - **Crusher** : Plantes → Bio Fuel
        - **Pressurized Reaction Chamber** : Bio Fuel + Eau + Hydrogen → Ethylene
        - **Gas-Burning Generator** : Ethylene → ~400 RF/t
        - Setup automatisable avec ferme de melons/cactus

    !!! warning "Fission Reactor"
        - Génère de la **chaleur** qui doit être évacuée
        - Nécessite **Fissile Fuel** (Uranium processing)
        - **PEUT MELTDOWN** si le refroidissement échoue!
        - Utiliser Sodium Cooling pour meilleure efficacité
        - Toujours avoir un système de sécurité (redstone control)

    !!! example "Fusion Reactor"
        - End-game absolu de Mekanism
        - Utilise **D-T Fuel** (Deuterium + Tritium)
        - Génère des **millions de RF/t**
        - Setup complexe : Laser → Hohlraum → Injection → Ignition
        - Deuterium : Électrolyse de l'eau lourde (Heavy Water)
        - Tritium : Lithium dans un Solar Neutron Activator

=== "Objets Importants"

    | Objet | Priorité | Description |
    |-------|:--------:|-------------|
    | Heat Generator | :zap: RUSH | Génération passive avec lave dessous |
    | Solar Generator | :star: | Énergie solaire basique (~50 RF/t) |
    | Advanced Solar Generator | :star: | Version améliorée du Solar |
    | Wind Generator | :star: | Production variable selon altitude |
    | Bio Generator | :star: | Utilise Bio Fuel (~200 RF/t) |
    | Gas-Burning Generator | :star::star: | Ethylene = ~400 RF/t |
    | Fission Reactor | :star::star: | ~500k RF/t (dangereux) |
    | Industrial Turbine | :star::star: | Convertit Steam en énergie |
    | Fusion Reactor | :star::star::star: | Millions de RF/t (end-game) |
    | Energy Cube | :star: | Stockage d'énergie (tiers) |

## Thermal Dynamos

[CurseForge](https://www.curseforge.com/minecraft/mc-mods/thermal-expansion) | [Documentation Officielle](https://teamcofh.com/docs/)

!!! success "Quick Start"
    1. Craft une **Steam Dynamo** (eau + charbon)
    2. Ajoute des **Transmission Coils** pour augmenter l'output
    3. Setup une **Magmatic Dynamo** avec lave du Nether
    4. Expérimente avec les autres types de Dynamos
    5. Utilise les **Augments** pour spécialiser tes Dynamos

=== "Tips"

    !!! tip "Types de Dynamos"
        - **Steam Dynamo** : Eau + Combustible solide (le plus basique)
        - **Magmatic Dynamo** : Lave (facile avec Nether)
        - **Compression Dynamo** : Fuel liquide (très efficace)
        - **Numismatic Dynamo** : Coins (fun mais niche)
        - **Lapidary Dynamo** : Gems = énergie
        - **Disenchantment Dynamo** : Recycle les items enchantés

    !!! tip "Augments Puissants"
        - **Transmission Coil** : +RF/t output (stackable!)
        - **Fuel Catalyzer** : +efficacité fuel
        - **Ignition Plugs** : +vitesse pour Compression Dynamo
        - **Boiler Conversion** : Produit Steam au lieu de RF

    !!! warning "Capacité vs Output"
        - Les Augments prennent des slots limités
        - Équilibrer entre efficacité et vitesse selon les besoins
        - Plusieurs petites Dynamos peuvent être mieux qu'une grosse

    !!! example "Setup Optimal"
        - **Compression Dynamo + Spark Plugs + 3 Transmission Coils** = **760 RF/t**
        - **Magmatic Dynamo** avec Fluid Duct vers Nether = énergie quasi-infinie
        - **Nullifier** pour éliminer les sous-produits indésirables

=== "Objets Importants"

    | Objet | Priorité | Description |

    | Steam Dynamo | :zap: RUSH | Dynamo basique (eau + charbon) |
    | Magmatic Dynamo | :star: | Utilise la lave (40-200 RF/t) |
    | Compression Dynamo | :star::star: | Fuel liquide (très efficace) |
    | Lapidary Dynamo | :star: | Gems → énergie |
    | Disenchantment Dynamo | | Recycle les enchantements |
    | Numismatic Dynamo | | Coins → énergie |
    | Transmission Coil | :zap: RUSH | Augmente l'output RF/t |
    | Fuel Catalyzer | :star: | Augmente l'efficacité du fuel |
    | Ignition Plugs | :star: | +vitesse Compression Dynamo |
    | Energy Cell | :star: | Stockage d'énergie Thermal |
    | Fluxduct | :star: | Câbles d'énergie configurables |

## Flux Networks

[CurseForge](https://www.curseforge.com/minecraft/mc-mods/flux-networks)

!!! success "Quick Start"
    1. Craft du **Flux Dust** (redstone + charbon dans le feu)
    2. Craft un **Flux Controller** pour gérer le réseau
    3. Craft des **Flux Plugs** (input) et **Flux Points** (output)
    4. Crée un réseau dans l'interface du Controller
    5. Connecte Plugs et Points au même réseau

=== "Tips"

    !!! tip "Architecture du Réseau"
        - **Flux Plugs** : Injectent l'énergie DANS le réseau
        - **Flux Points** : Extraient l'énergie DU réseau
        - **Flux Controller** : Gère le réseau (un seul nécessaire)
        - Tout fonctionne **sans fil et cross-dimension**

    !!! tip "Priorités"
        - Configurer les **priorités** pour distribution intelligente
        - Haute priorité = reçoit l'énergie en premier
        - Utile pour prioriser machines critiques vs stockage

    !!! warning "Flux Dust"
        - Jeter **Redstone + Charbon** dans le feu pour obtenir Flux Dust
        - Automatisable avec droppers/dispensers
        - Base de tous les crafts Flux Networks

    !!! example "Setup Optimal"
        - **Flux Storage** dans le réseau comme buffer
        - **Wireless Charging** pour charger l'inventaire du joueur
        - Un seul Plug sur ta source principale, Points partout ailleurs
        - **Chunk loading** nécessaire pour cross-dimension

=== "Objets Importants"

    | Objet | Priorité | Description |
    |-------|:--------:|-------------|
    | Flux Dust | :zap: RUSH | Matériau de base (redstone + charbon dans feu) |
    | Flux Controller | :zap: RUSH | Gestion centralisée du réseau |
    | Flux Plug | :star: | Input d'énergie vers le réseau |
    | Flux Point | :star: | Output d'énergie depuis le réseau |
    | Flux Storage | :star::star: | Stockage dans le réseau (buffer) |
    | Flux Configurator | :star: | Copie les configurations |

=== "Guides"

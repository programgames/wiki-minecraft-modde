## Progression Recommandée

!!! tip "Early Game"
    - :zap: **[RUSH]** **Water Candle** de Cyclic (augmente spawn rate en zone)
    - :zap: **[RUSH]** **Spikes** de Mini Utilities ou Dark Utilities (player kill = XP + rare loot)
    - :zap: **[RUSH]** **Mob Catcher** de Mob Grinding Utils pour capturer des mobs
    - **Cursed Earth** pour spawn massif si disponible

!!! note "Mid Game"
    - Développer les **Spawners Apotheosis** avec modifiers (Sugar, Clock, Spider Eye)
    - **Mob Grinding Utils** Mob Masher + Absorption Hopper pour full automation
    - Commencer les **Data Models** de Hostile Neural Networks

!!! success "Late Game"
    - **Spawners Apotheosis** maxés avec Echoing (Nether Star) pour double loot
    - **Hostile Neural Networks** Self-Aware models pour simulation passive
    - **Industrial Foregoing** Mob Duplicator + Mob Crusher pour spawning infini

## Synergies entre Mods

!!! abstract "Combinaisons Puissantes"

    **Apotheosis + Mob Grinding Utils**
    :   Spawner modifié avec Mob Masher enchantable (Looting V) = drops maximisés

    **Industrial Foregoing + Apotheosis**
    :   Mob Duplicator depuis Safari Net + Mob Crusher = farm automatique infinie

    **Hostile Neural Networks + Loot Fabricator**
    :   Data Model Self-Aware + Loot Fabricator = drops rares sans tuer de mobs

    **Mega Torch + Cursed Earth**
    :   Mega Torch bloque spawns partout + Cursed Earth dans la farm = spawns concentrés

    **Cyclic Water Candle + Spawner**
    :   Water Candle augmente le spawn rate même des spawners modifiés

    **Diamond Spike + Absorption Hopper**
    :   Player kill pour XP et rare loot + collecte automatique

## Apotheosis

[CurseForge](https://www.curseforge.com/minecraft/mc-mods/apotheosis) | [Wiki](https://github.com/Shadows-of-Fire/Apotheosis/wiki)

!!! success "Quick Start"
    1. Trouve un **Spawner** vanilla dans un donjon ou mineshaft
    2. Applique du **Sugar** pour réduire le délai minimum (-5 ticks par sugar)
    3. Applique une **Clock** pour réduire le délai maximum (-5 ticks par clock)
    4. Applique une **Fermented Spider Eye** pour +1 mob par spawn
    5. Utilise **Cobweb** pour No AI (mobs immobiles = farm facile)

=== "Tips"

    !!! tip "Modifiers de Base"
        - **Sugar** et **Clock** se stackent - applique-en beaucoup pour spawns très rapides
        - **Fermented Spider Eye** augmente le nombre de mobs par spawn (+1 par item)
        - **Comparator** ajoute le contrôle Redstone au spawner
        - Shift+clic droit avec un item vide pour voir les stats du spawner

    !!! tip "Modifiers Avancés"
        - **Glowstone** permet au spawner d'ignorer la lumière
        - **Cobweb** donne No AI aux mobs (ils ne bougent plus)
        - Combiner No AI avec des Spikes ou Mob Masher pour farm passive

    !!! warning "Attention"
        - Les spawners ne peuvent **pas être récupérés** sans Silk Touch (ou mods)
        - Certains modpacks désactivent le minage de spawners
        - **Echoing** (Nether Star) coûte très cher mais double le loot

    !!! example "Configuration Optimale"
        | Modifier | Quantité | Effet |
        |----------|----------|-------|
        | Sugar | Max | Délai minimum très court |
        | Clock | Max | Délai maximum très court |
        | Fermented Spider Eye | 4-6 | 4-6 mobs par spawn |
        | Cobweb | 1 | No AI |
        | Glowstone | 1 | Ignore lumière |
        | Nether Star | 1 | Echoing (double loot) |

=== "Objets Importants"

    | Objet | Priorité | Description |
    |-------|:--------:|-------------|
    | Sugar | :zap: RUSH | -5 ticks délai minimum par item |
    | Clock | :zap: RUSH | -5 ticks délai maximum par item |
    | Fermented Spider Eye | :star: | +1 mob par spawn |
    | Cobweb | :star: | No AI (mobs immobiles) |
    | Glowstone | :star: | Ignore la lumière |
    | Comparator | | Redstone control |
    | Nether Star | | Echoing (double loot) - très cher |
    | Ender Lead | | Déplace les Endermen sans TP |
    | Spawner Silk Touch | | Pour récupérer les spawners (si disponible) |

## Industrial Foregoing

[CurseForge](https://www.curseforge.com/minecraft/mc-mods/industrial-foregoing) | [Wiki](https://ftb.fandom.com/wiki/Industrial_Foregoing)

!!! success "Quick Start"
    1. Capture un mob avec le **Mob Imprisonment Tool**
    2. Place le tool dans un **Mob Duplicator** avec de l'énergie
    3. Connecte un **Mob Crusher** à côté de la zone de spawn
    4. Collecte l'**Essence** (XP liquide) générée par le Crusher
    5. Ajoute des **Range Addons** pour agrandir les zones de travail

=== "Tips"

    !!! tip "Mob Crusher"
        - Fait **300 dégâts par tick** (tue tout sauf Wither en un coup)
        - **Ne tue pas les bébés** - pratique pour breeding
        - Mode **Looting** : désactive Essence mais donne loot bonus random
        - Génère de l'**Essence** (XP liquide) par défaut

    !!! tip "Mob Duplicator"
        - Spawne des mobs depuis un **Safari Net** ou **Mob Imprisonment Tool**
        - Consomme de l'**Essence** pour spawner
        - Très efficace combiné avec le Mob Crusher (loop d'Essence)
        - **Range Addon** pour agrandir la zone de spawn

    !!! warning "Ressources Requises"
        - Les machines nécessitent beaucoup d'**énergie RF**
        - Le Mob Duplicator consomme de l'**Essence** par spawn
        - Prévoir un bon setup d'énergie avant de commencer

    !!! example "Setup Complet"
        ```
        [Mob Duplicator] → [Zone de Spawn] → [Mob Crusher]
              ↑                                    ↓
              ←←←←←← [Essence Tank] ←←←←←←←←←←←←←←
        ```

=== "Objets Importants"

    | Objet | Priorité | Description |
    |-------|:--------:|-------------|
    | Mob Imprisonment Tool | :zap: RUSH | Capture n'importe quel mob |
    | Mob Crusher | :star: | Tue et génère Essence (300 dmg/tick) |
    | Mob Duplicator | :star: | Spawne des mobs depuis Safari Net |
    | Mob Slaughter Factory | :star: | Génère Pink Slime et Meat |
    | Range Addon | :star: | Augmente la zone des machines |
    | Animal Rancher | | Collecte laine, lait, oeufs |
    | Animal Feeder | | Nourrit automatiquement |
    | Animal Baby Separator | | Sépare bébés des adultes |
    | Essence (fluide) | | XP liquide, stockable en tanks |

=== "Guides"

    - [Wiki Industrial Foregoing](https://ftb.fandom.com/wiki/Industrial_Foregoing)
    - [SkyFactory Guide](https://skyfactory-4.fandom.com/wiki/Industrial_Foregoing)

## Hostile Neural Networks

[CurseForge](https://www.curseforge.com/minecraft/mc-mods/hostile-neural-networks) | [Wiki](https://github.com/Shadows-of-Fire/Hostile-Neural-Networks/wiki)

!!! success "Quick Start"
    1. Craft un **Data Model** vide et place-le dans ton inventaire
    2. Tue des mobs du type voulu - le modèle apprend automatiquement
    3. Le modèle monte en tier : Faulty → Basic → Advanced → Superior → Self-Aware
    4. Place le modèle dans une **Simulation Chamber** avec de l'énergie
    5. Récupère les **Predictions** et utilise le **Loot Fabricator** pour les drops

=== "Tips"

    !!! tip "Système de Tiers"
        - **Faulty** : Production très lente, beaucoup d'échecs
        - **Basic** : Production lente
        - **Advanced** : Production moyenne
        - **Superior** : Bonne production
        - **Self-Aware** : Production maximale, aucun échec

    !!! tip "Optimisation"
        - Plus le tier est haut, moins la chambre consomme d'énergie
        - **Self-Aware** = efficacité maximale, rush ce tier
        - Utiliser plusieurs **Simulation Chambers** en parallèle
        - Les **Predictions** stackent à 64

    !!! warning "Attention"
        - Le modèle doit être dans l'**inventaire** pour apprendre (pas hotbar seule)
        - Chaque type de mob a besoin de son propre Data Model
        - Les boss peuvent être plus difficiles à modéliser

    !!! example "Drops Rentables"
        | Mob | Drop Notable | Utilité |
        |-----|--------------|---------|
        | Wither Skeleton | Wither Skull | Spawn Wither |
        | Blaze | Blaze Rod | Brewing, fuel |
        | Enderman | Ender Pearl | Téléportation |
        | Ghast | Ghast Tear | Potions |
        | Shulker | Shulker Shell | Shulker Boxes |

=== "Objets Importants"

    | Objet | Priorité | Description |
    |-------|:--------:|-------------|
    | Data Model | :zap: RUSH | Un par type de mob, apprend en tuant |
    | Simulation Chamber | :star: | Génère des Predictions depuis les modèles |
    | Loot Fabricator | :star: | Convertit Predictions en drops réels |
    | Prediction Matrix | :star: | Matériau de craft pour les machines |
    | Deep Learner | | Stocke plusieurs Data Models |

=== "Guides"

    - [HNN Wiki](https://github.com/Shadows-of-Fire/Hostile-Neural-Networks/wiki)
    - [Getting Started Guide](https://www.youtube.com/watch?v=hnn-guide)

## Mob Grinding Utils

[CurseForge](https://www.curseforge.com/minecraft/mc-mods/mob-grinding-utils) | [Wiki](https://github.com/vadis365/Mob-Grinding-Utils/wiki)

!!! success "Quick Start"
    1. Craft un **Mob Masher** et place-le face aux mobs
    2. **Enchante le Mob Masher** avec Looting et Sharpness
    3. Ajoute un **Absorption Hopper** pour collecter items ET XP
    4. Utilise des **Fans** avec upgrades pour pousser les mobs vers le Masher
    5. Place un **XP Tap** sur l'Absorption Hopper pour extraire l'XP en fluide

=== "Tips"

    !!! tip "Mob Masher"
        - Le Mob Masher est **enchantable** sur une table d'enchantement!
        - **Looting III** = plus de drops
        - **Sharpness V** = plus de dégâts
        - **Beheading** = plus de têtes de mobs

    !!! tip "Système de Collecte"
        - L'**Absorption Hopper** collecte **items ET XP** simultanément
        - **XP Tap** sur le hopper pour extraire l'XP en fluide
        - **Tank** pour stocker l'XP liquide
        - **XP Solidifier** pour convertir en bouteilles ou blocs

    !!! warning "Attention"
        - Les **Fans** nécessitent de l'énergie RF
        - L'**Ender Inhibitor** a un rayon de 8 blocs seulement
        - Les **Spikes** ont différents tiers - Diamond = player kill

    !!! example "Setup Anti-Enderman"
        - **Ender Inhibitor** empêche les Endermen de TP (rayon 8)
        - Placer plusieurs inhibitors pour couvrir toute la farm
        - Combiner avec Fans pour les pousser vers le Mob Masher

=== "Objets Importants"

    | Objet | Priorité | Description |
    |-------|:--------:|-------------|
    | Mob Masher | :zap: RUSH | Tue automatiquement, enchantable! |
    | Absorption Hopper | :zap: RUSH | Collecte items ET XP |
    | Fan | :star: | Pousse les mobs (configurable, upgradable) |
    | XP Tap | :star: | Extrait l'XP du hopper en fluide |
    | Tank | :star: | Stocke l'XP liquide |
    | Spikes | :star: | Tue les mobs (Diamond = player kill) |
    | Ender Inhibitor | :star: | Empêche les Endermen de TP (rayon 8) |
    | Mob Swab | | Collecte les gènes pour breeding |
    | GM Chicken Feed | | Breeding de chickens modifiés |
    | XP Solidifier | | Convertit XP en items |

=== "Guides"

    - [MGU Wiki](https://github.com/vadis365/Mob-Grinding-Utils/wiki)
    - [Complete Guide Video](https://www.youtube.com/watch?v=mgu-guide)

## Mods Secondaires

??? note "Deep Mob Learning"

    [CurseForge](https://www.curseforge.com/minecraft/mc-mods/deep-mob-learning)

    !!! success "Quick Start"
        1. Craft un **Data Model** et place-le dans ton inventaire
        2. Tue des mobs du type correspondant pour l'entraîner
        3. Place le modèle dans une **Simulation Chamber**
        4. Récupère **Living Matter** et **Pristine Matter**
        5. Utilise l'**Extraction Chamber** ou **Loot Fabricator** pour les drops

    === "Tips"

        !!! tip "Système de Tiers"
            - Tiers : Faulty → Basic → Advanced → Superior → Self-Aware
            - Plus le tier est haut, plus la production est rapide
            - **Pristine Matter** est le drop rare et précieux

        !!! tip "Trial System"
            - Le **Trial Keystone** permet de combattre des mobs simulés
            - Récompenses uniques et **Glitch Armor** materials
            - **Glitch Armor** est l'armure end-game du mod

    === "Objets Importants"

        | Objet | Priorité | Description |
        |-------|:--------:|-------------|
        | Data Model | :star: | Apprend d'un type de mob |
        | Simulation Chamber | :star: | Simule les mobs pour drops |
        | Extraction Chamber | :star: | Extrait Living/Pristine Matter |
        | Loot Fabricator | :star: | Convertit la matière en drops |
        | Trial Keystone | | Lance des trials pour récompenses |
        | Glitch Armor | | Armure end-game puissante |

??? note "Capture de Mobs"

    | Item | Mod | Notes |
    |------|-----|-------|
    | Mob Imprisonment Tool | Industrial Foregoing | Le plus commun, fonctionne sur tout |
    | Safari Net | MFR/Industrial | Variantes single-use et réutilisables |
    | Lasso | Extra Utilities | Simple et efficace |
    | Monster Ball | Cyclic | Facile à craft |
    | Mob Catcher | Mob Grinding Utils | Bonne option early |
    | Soul Vial | Ender IO | Classique |
    | Quantum Catcher | Forbidden Arcanus | Capture tout, même les boss |

    !!! tip "Tips de Capture"
        - Capturer un mob **nommé** garde le nom
        - Certaines captures gardent les **stats du mob** (HP, etc.)
        - Les **boss** ne peuvent souvent pas être capturés (sauf Quantum Catcher)

??? note "Augmenter le Spawn Rate"

    | Item | Mod | Effet |
    |------|-----|-------|
    | Water Candle | Cyclic | :zap: **[RUSH]** Augmente spawn rate en zone |
    | Monolith of Spawning | Ender Utilities | :zap: **[RUSH]** Spawn boost massif |
    | Cursed Earth | Extra Utilities | Spawn massif sur ce bloc |
    | Mega Torch | Torch Master | Bloque spawn ailleurs (force spawn à un endroit) |

    !!! tip "Stratégie"
        - Utiliser **Mega Torch** pour bloquer les spawns partout sauf dans ta farm
        - Combiner plusieurs méthodes pour maximiser le spawn
        - **Cursed Earth** + zone fermée = spawn très rapide

??? note "Spikes et Pièges"

    | Diamond Spike | Mini Utilities | :star: Player kill (XP + rare loot) |

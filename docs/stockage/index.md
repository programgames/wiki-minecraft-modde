## Progression Recommandée

!!! tip "Early Game"
    - :zap: **[RUSH]** **Functional Storage** pour les drawers de masse
    - :zap: **[RUSH]** **Sophisticated Backpacks** pour le stockage portable
    - :zap: **[RUSH]** **Dank Storage** pour le stockage compact
    - :zap: **[RUSH]** **Ender Storage** pour le partage cross-dimension
    - :zap: **[RUSH]** **RFTools Storage Scanner** comme mini-AE2
    - Explorer **Occultism** pour le stockage dimensionnel gratuit

!!! note "Mid Game"
    - **Simple Storage Network** pour accès centralisé aux coffres
    - Transition vers **AE2** ou **Refined Storage**
    - **Storage Bus** sur les drawers existants pour intégration
    - Setup de base pour l'autocrafting

!!! success "Late Game"
    - **AE2 complet** avec tous les addons (Mega Cells, Advanced AE)
    - **QIO de Mekanism** pour accès wireless illimité
    - Autocrafting massif avec CPU multi-core
    - Integration complète avec toutes les machines

## Applied Energistics 2

[CurseForge](https://www.curseforge.com/minecraft/mc-mods/applied-energistics-2) | [Guide Officiel](https://guide.appliedenergistics.org/)

!!! success "Quick Start"
    1. Mine des **Certus Quartz** et **Meteorites** pour les presses
    2. Craft un **Charger** et charge du Certus en **Charged Certus**
    3. Crée des **Fluix Crystals** en jetant Charged Certus + Redstone + Nether Quartz dans l'eau
    4. Craft un **Inscriber** pour fabriquer les processeurs
    5. Construit ton premier **ME Controller** + **ME Drive** + **Terminal**

### Addons

- [Advanced AE](https://www.curseforge.com/minecraft/mc-mods/advancedae) : Export bus amélioré, tour à craft
- [Mega Cells](https://www.curseforge.com/minecraft/mc-mods/mega-cells) : Disques massifs (256k+)
- [AE2 Wireless Terminals](https://www.curseforge.com/minecraft/mc-mods/applied-energistics-2-wireless-terminals) : Accès sans fil
- [AE Infinity Booster](https://www.curseforge.com/minecraft/mc-mods/aeinfinitybooster) : Range wireless infinie
- [Applied Mekanistics](https://www.curseforge.com/minecraft/mc-mods/applied-mekanistics) : Stockage gaz/slurry
- [ME Requester](https://www.curseforge.com/minecraft/mc-mods/merequester) : Auto-stock automatique

=== "Tips"

    !!! tip "Channels & Cables"
        - **Dense Smart Cable** = 32 channels (vs 8 pour câble normal)
        - **P2P Tunnels** transportent 32 channels dans un seul câble
        - **Quartz Fiber** transfère l'énergie sans connecter les channels
        - Les **Molecular Assemblers ne consomment pas de channel** mais en transfèrent 8

    !!! tip "Autocrafting"
        - **Pattern Provider entouré de 6 Molecular Assemblers** = setup optimal
        - **Acceleration Cards** dans les Assemblers = plus de vitesse
        - **Blocking Mode** pour machines qui ne prennent qu'un batch (ex: Energizing Orb)
        - **Substitution** dans les patterns pour recettes avec variantes

    !!! warning "Attention"
        - **Crystal Growth Accelerators** autour des cristaux = x17 plus rapide
        - Le Controller est obligatoire pour plus de 8 channels
        - Ne pas oublier l'**Energy Acceptor** pour alimenter le réseau

    !!! example "Setup Optimal"
        - **Formation Plane** pour placer des blocs automatiquement
        - **Level Emitter** pour redstone quand un item atteint un seuil
        - **Storage Bus sur chest** comme buffer temporaire pour les crafts

## Refined Storage

[CurseForge](https://www.curseforge.com/minecraft/mc-mods/refined-storage) | [Guide Officiel](https://refinedmods.com/refined-storage/)

!!! success "Quick Start"
    1. Craft un **Controller** (cœur du réseau)
    2. Craft un **Disk Drive** et des **Storage Disks**
    3. Connecte avec des **Cables** et ajoute un **Grid**
    4. Craft un **Crafting Grid** pour le crafting intégré
    5. Ajoute des **Crafters** + **Patterns** pour l'autocrafting

### Addons

- [Extra Disks](https://www.curseforge.com/minecraft/mc-mods/extra-disks) : Disques plus gros
- [Cable Tiers](https://www.curseforge.com/minecraft/mc-mods/cable-tiers) : Câbles plus rapides
- [Extra Storage](https://www.curseforge.com/minecraft/mc-mods/extrastorage) : Plus de stockage

### Différences avec AE2

| Aspect | Refined Storage | AE2 |
|--------|:---------------:|:---:|
| Channels | :x: Non | :white_check_mark: Oui (8/32) |
| Complexité | Simple | Complexe |
| Autocrafting | Direct | Plus optimisable |

=== "Tips"

    !!! tip "Configuration"
        - **Pas de système de channels** = plus simple à configurer
        - Le **Controller** est le cœur (un seul par réseau)
        - Les câbles colorés permettent de **séparer les réseaux**
        - **Wireless Transmitter** pour accès sans fil

    !!! tip "Autocrafting"
        - **Crafter** + **Pattern** pour l'autocrafting
        - Plusieurs Crafters peuvent être chaînés
        - **Processing patterns** pour machines externes

    !!! example "Setup Rapide"
        - **External Storage** connecte des inventaires externes
        - **Importer/Exporter** pour transfert automatique
        - **Detector** émet redstone selon le stock

## Functional Storage

[CurseForge](https://www.curseforge.com/minecraft/mc-mods/functional-storage)

!!! success "Quick Start"
    1. Craft des **Drawers** (1x1, 2x2, ou 4x4)
    2. Place un **Storage Controller** près des drawers
    3. Utilise le **Linking Tool** : Left-click controller, Right-click drawer
    4. Ajoute des **Upgrades** pour augmenter la capacité
    5. Utilise un **Compacting Drawer** pour ingot/block/nugget automatique

=== "Tips"

    !!! tip "Controller & Linking"
        - **Linking Tool** : Left-click controller → Right-click drawer pour lier
        - Range par défaut : **8 blocs** du controller (upgradable)
        - Le controller donne accès centralisé à tous les drawers liés

    !!! tip "Upgrades Puissants"
        - Les upgrades donnent des boosts **plus importants que Storage Drawers**
        - **Ender Drawers** synchronisés cross-dimension
        - **Void Upgrade** détruit le surplus (évite overflow)

    !!! warning "Capacité"
        - Plus de slots (2x2, 4x4) = **moins de capacité par slot**
        - 1x1 Drawer est meilleur pour les items très nombreux

    !!! example "Automation"
        - **Collector Upgrade** : Vacuum items par terre
        - **Puller/Pusher Upgrade** : Transfert automatique vers/depuis inventaires
        - Compatible avec **AE2 Storage Bus** pour intégration

=== "Objets Importants"

    | Objet | Priorité | Description |
    |-------|:--------:|-------------|
    | 1x1 Drawer | :zap: RUSH | Un slot, capacité maximale par item |
    | 2x2 / 4x4 Drawer | - | Plus de slots, moins de capacité chacun |
    | Compacting Drawer | - | Stocke ingot + block + nugget auto |
    | Storage Controller | - | Accès centralisé à tous les drawers liés |
    | Linking Tool | :zap: RUSH | Lie les drawers au controller |
    | Ender Drawer | | Synchronisé cross-dimension |
    | Void Upgrade | - | Détruit le surplus |
    | Collector Upgrade | | Vacuum items par terre |
    | Puller/Pusher Upgrade | | Transfert automatique |
    | Storage Upgrade (tiers) | - | Augmente la capacité |

=== "Guides"

    - [Functional Storage Wiki](https://github.com/InnovativeOnlineIndustries/Functional-Storage/wiki)

## Sophisticated Backpacks

[CurseForge](https://www.curseforge.com/minecraft/mc-mods/sophisticated-backpacks)

!!! success "Quick Start"
    1. Craft un **Leather Backpack** basique
    2. Upgrade vers **Iron → Gold → Diamond → Netherite**
    3. Ajoute des **Upgrade Slots** pour plus de fonctionnalités
    4. Installe des **Pickup/Filter Upgrades** pour collecte automatique
    5. Utilise le **Feeding Upgrade** pour manger automatiquement

=== "Tips"

    !!! tip "Upgrades"
        - Les backpacks peuvent être **tiered up** sans perdre le contenu
        - Chaque tier ajoute des slots de stockage ET d'upgrade
        - Peut être ouvert **depuis l'inventaire** sans le placer

    !!! tip "Upgrades Essentiels"
        - **Feeding Upgrade** : Mange automatiquement depuis le backpack
        - **Magnet Upgrade** : Attire les items
        - **Crafting Upgrade** : Table de craft portable

    !!! warning "Curios Integration"
        - Équiper en **slot Curio** pour accès permanent
        - Keybind pour ouvrir directement

    !!! example "Automation"
        - **Pickup Upgrade** : Collecte certains items directement
        - **Filter Upgrade** : Filtre ce qui entre
        - **Void Upgrade** : Détruit les items spécifiés

=== "Objets Importants"

    | Objet | Priorité | Description |
    |-------|:--------:|-------------|
    | Leather Backpack | :zap: RUSH | Premier backpack craftable |
    | Iron/Gold/Diamond/Netherite | - | Tiers supérieurs |
    | Feeding Upgrade | - | Auto-feed depuis le backpack |
    | Pickup Upgrade | - | Collecte items au sol |
    | Filter Upgrade | | Filtre les entrées |

            - **Iron** : Encore plus
            - **Diamond** : Maximum de capacité par bloc

            - Utilise plutôt plusieurs petits que un énorme

        3. Place un **Diamond** sur le loquet pour rendre le coffre privé
        4. Craft un **Ender Pouch** pour accès portable

    === "Tips"

        !!! tip "Code Couleur"
            - **3 couleurs** configurables par coffre
            - Synchronisés **cross-dimension** instantanément
            - Parfait pour **transfert entre bases**

        !!! tip "Privé vs Public"
            - **Diamond sur le loquet** = coffre privé (lié au joueur)
            - Sans diamond = accessible par tous avec le même code

        - **Ender Tanks** pour les fluides (même système de couleurs)
        - Automatisable avec hoppers/pipes

    === "Objets Importants"

        | Objet | Priorité | Description |
        |-------|:--------:|-------------|
        | Ender Chest | :zap: RUSH | Coffre partagé par couleurs |
        | Ender Tank | | Tank partagé par couleurs |
        | Ender Pouch | - | Accès portable à un Ender Chest |

    Progression Recommandée¶

    Early Game

    • [RUSH] Botany Pots pour le farming passif en 1x1
    • [RUSH] Sprinkler de Cyclic pour accélérer les crops
    • [RUSH] Scythe de Mystical Agriculture pour harvest AOE
    • Farmer's Delight pour la nourriture de qualité avec buffs

    Mid Game

    • Développer Mystical Agriculture tier par tier
    • Productive Bees pour les ressources automatiques
    • Industrial Foregoing Plant Gatherer/Sower pour full automation

    Late Game

    • Mystical Agriculture Tier 6 (Supremium) pour toutes les ressources
    • Productive Bees avec génétique optimisée (Very High productivity)
    • Combiner Botany Pots + Compacting Drawers pour auto-compression

    Synergies entre Mods¶

    Mystical Agriculture¶

    CurseForge | Wiki Officiel

    Quick Start

    1. Craft un Inferium Seed et plante-le sur du farmland
    2. Récolte l'Inferium Essence et combine 4→1 pour tier up
    3. Craft une Infusion Crystal pour créer des seeds de ressources
    4. Place des Growth Accelerators sous le farmland (stackable 64x)
    5. Craft une Scythe pour harvest en AOE (3x3 à 7x7)

    Addons¶

    • Mystical Agradditions : Tiers supplémentaires, crops pour dragons
    • Mystical Agriculture Tiered Crystals

    Croissance & Farmland

    • Les Inferium Seeds sur Essence Farmland ont +10% chance de drop une seed bonus
    • Planter sur farmland du même tier = +10% supplémentaire de seeds
    • Les crops ignorent la lumière et n'ont pas besoin d'eau
    • Les Growth Accelerators se stackent sous le farmland (max 64 en dessous)

    Automation & Crafting

    • Automatiser le Watering Can avec Modular Router ou fake player
    • Utiliser les crafters de RFTools pour auto-craft les essences
    • Logistical Sorter de Mekanism pour trier les essences automatiquement
    • Combiner avec Industrial Foregoing Plant Gatherer pour full automation

    Attention

    • Inferium essence se combine 4→1 en essence du tier supérieur
    • Le Soulium Dagger augmente les drops de Mob Chunks - essentiel pour les seeds de mobs

    End-Game

    • Supremium Tools sont réparables avec Supremium Essence
    • Les Awakened Supremium tools (avec Agradditions) sont encore plus puissants
    • Armure Supremium avec augments (vol, vitesse) via la Tinkering Table
    Objet Priorité Description
    Scythe RUSH Harvest en AOE (3x3 à 7x7 selon tier)
    Inferium Seeds Base de toutes les essences
    Essence Farmland Sol qui boost les drops (+10% Inferium, +50% Supremium)
    Growth Accelerator Accélère la croissance (stackable jusqu'à 64 sous le sol)
    Infusion Crystal Requis pour craft les seeds (1000 uses)
    Master Infusion Crystal Version durabilité infinie
    Soul Jar Stocke les âmes de mobs pour seeds

    Productive Bees¶

    CurseForge | Documentation | All The Guides

    Quick Start

    1. Utilise un Nest Locator pour trouver les nids sauvages
    2. Capture des abeilles avec une Bee Cage (stack à 64)
    3. Place-les dans une Advanced Beehive avec des upgrades
    4. Construit une Centrifuge pour extraire les produits des combs
    5. Utilise le Big Book of Bees ou JEI pour les recettes de breeding

    Génétique & Breeding

    • L'abeille de gauche dans le breeding = gènes prioritaires
    • Les bébés ont au minimum les gènes de l'abeille principale (jamais en dessous)
    • Squash une abeille pour obtenir ses gènes (% partiel de chaque gène)
    • Combiner gènes + Honey Treat puis right-click sur une abeille pour transférer les traits

    Production & Automation

    • Vérifier les flower requirements dans JEI (right-click ou U sur l'abeille)
    • Simulation upgrades dans l'Advanced Beehive = production sans sortir
    • Filtering Upgrade dans la ruche pour ne garder que certains combs
    • Comb Blocks + Saw = conversion en masse des combs

    Abeilles Spéciales

    • Farmer Bees récoltent les crops, combiner avec Hoarder Bee pour auto-farm
    • Les Wither Bee et Dragon Bee sont les plus rentables mais difficiles à obtenir

    Génétique Optimale

    Farmer's Delight¶

    CurseForge | Wiki

    Quick Start

    1. Craft une Cutting Board et un Knife
    2. Découpe du blé sur la cutting board pour obtenir du Straw
    3. Craft un Cooking Pot et place-le sur un feu
    4. Crée des plats avec buffs comme le Steak and Potatoes
    5. Transforme le Straw en Organic Compost puis en Rich Soil

    Addons¶

    • Cooking for Blockheads
    • Croptopia
    • Corn Delight
    • End's Delight
    • Delightful

    Cuisine & Buffs

    • La Cutting Board + Knife débloque beaucoup de recettes de cuisine
    • Les plats cuisinés donnent des buffs en plus de la saturation
    • Nourished réduit la perte de faim, Comfort accélère la regen naturelle
    • Skillet permet de cuisiner en main droite près d'un feu

    Rich Soil & Farming

    • Straw (drop du wheat) → compost → Organic Compost → Rich Soil
    • Rich Soil applique automatiquement du bone meal aux crops adjacents
    • Très efficace combiné avec d'autres mods de farming

    Décorations Utiles

    • Le Canvas permet de faire des sacs et décorations
    • Tatami blocks empêchent le spawn de mobs (utile pour builds)
    • Rope pour la décoration et l'escalade

    Botany Pots¶

    CurseForge

    Quick Start

    1. Craft un Botany Pot basique
    2. Place un sol dans le pot (dirt, farmland, soul sand, etc.)
    3. Place une seed ou plante dans le pot
    4. Attend la croissance et récolte, ou utilise un Hopper Botany Pot
    5. Pour Mystical Agriculture, utilise du Essence Farmland comme sol

    Utilisation Basique

    • Les pots peuvent cultiver presque n'importe quelle plante
    • Utiliser un hopper dessous pour collecter automatiquement
    • Les sols améliorés accélèrent la croissance
    • Peut cultiver des crops de Mystical Agriculture

    Automation

    • Hopper Botany Pot exporte automatiquement vers un inventaire
    • Combiner avec Compacting Drawers pour auto-compression des essences
    • Très compact : farming efficace en 1x1

    Sols Recommandés

    • Farmland : Pour la plupart des crops
    • Essence Farmland : Pour Mystical Agriculture (bonus drops)
    • Soul Sand : Pour Nether Wart
    • End Stone : Pour Chorus Fruit
    Objet Priorité Description
    Botany Pot RUSH Pot de base pour cultiver
    Hopper Botany Pot Exporte automatiquement les drops
    Différents sols Terre, farmland, soul sand, etc.

    Mods Secondaires¶

    Industrial Foregoing

    CurseForge | Guide SkyFactory

    Quick Start

    1. Craft un Plant Sower pour planter automatiquement
    2. Craft un Plant Gatherer pour récolter automatiquement
    3. Ajoute un Plant Fertilizer pour accélérer la croissance
    4. Utilise Hydroponic Bed pour une croissance ultra-rapide
    5. Connecte le tout avec des conveyors ou pipes

    Farming Automation

    • Plant Sower + Plant Gatherer = farming automatique complet
    • Plant Fertilizer applique bone meal automatiquement
    • Hydroponic Bed = sol très rapide pour les crops

    Autres Machines Utiles

    • Animal Rancher collecte laine, lait, etc.
    • Animal Baby Separator sépare les bébés des adultes
    • Meat Feeder pour manger automatiquement

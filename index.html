    ✅ Quick Start Checklist¶

    5 Premières Choses à Faire dans Tout Modpack

    1. 📦 Configurer JEI/REI - Apprenez les raccourcis (R pour recettes, U pour usages, / pour chercher)
    2. 🏠 Base temporaire - Construisez un abri sécurisé avec lit, coffres et établi
    3. ⛏️ Outils Tinker's/Silent Gear - Créez des outils évolutifs dès que possible
    4. 🔋 Première source d'énergie - Générateur basique (Mekanism, Thermal ou Create)
    5. 📚 Lire les quêtes - Consultez le livre de quêtes pour comprendre la progression

    Progression Recommandée¶

    Early Game

    • [RUSH] Commencer Ars Nouveau pour la téléportation et le farming
    • [RUSH] Botania avec Endoflames pour le mana passif
    • [RUSH] Eternal Stella (Forbidden & Arcanus) sur ton meilleur outil
    • Explorer les donjons pour trouver des artefacts magiques

    Mid Game

    • Développer Blood Magic pour les rituels puissants
    • Occultism pour le stockage dimensionnel
    • Spirit pour les mob farms avec Soul Cages

    Late Game

    • Combiner Ars Nouveau + Apotheosis pour des enchantements puissants
    • Automatisation complète avec Occultism
    • Armures magiques end-game (Iron's Spells, Blood Magic Living Armor)

    Synergies entre Mods¶

    Combinaisons Puissantes

    Ars Nouveau + Occultism (via Ars Ocultas)
    Les esprits Occultism peuvent être invoqués avec des sorts Ars
    Ars Nouveau + Create (via Ars Creo)
    Automatisation des recettes Ars avec Create
    Botania + Apotheosis
    Mana Enchanting pour des enchantements surpuissants
    Blood Magic + Spirit
    Well of Suffering sur un Soul Cage = LP infini
    Nature's Aura + Mystical Agriculture
    Farming Stencil + Inferium = croissance ultra-rapide

    Ars Nouveau¶

    CurseForge | Ars.Guide | Wiki Fandom

    Quick Start

    1. Craft un Worn Notebook (Book + Lapis)
    2. Plante des Sourceberries près de ta base
    3. Construit un Enchanting Apparatus et une Scribes Table
    4. Crée ton premier spell avec Touch + Break
    5. Invoque un Starbuncle pour transporter tes items

    Addons¶

    • Ars Creo : Intégration avec Create
    • Ars Elemental : Nouveaux éléments et glyphes
    • Ars Ocultas : Intégration Occultism
    • All The Arcanist Gear : Équipement supplémentaire

    Génération de Source

    • [RUSH] Planter des Sourceberries près de ta base pour génération passive
    • Source Jars en ligne se transfèrent le Source automatiquement
    • Utiliser Siphon pour récupérer du mana sur les hits

    Familiars & Servants

    • Starbuncles transportent les items - donne-leur un Item Scroll pour spécifier quoi
    • Wixies pour l'auto-crafting - lie une Wixie à un chaudron
    • Drygmy farms sont OP - Drygmies près de mobs passifs = drops automatiques
    • Whirlisprig pour le bois - auto-plante et fait pousser les arbres

    Attention

    • Amplify a été nerf en 2024 : Le scaling est maintenant exponentiel
    • Le mana cost augmente rapidement avec les glyphes

    Spells Recommandés

    • Combat : Orb + AOE devastation + knockback
    • Mining : Touch + Break + AOE + Extract
    • Téléportation : Blink + Self

    Blood Magic¶

    CurseForge

    Quick Start

    1. Craft un Blood Altar (pierre + diamond)
    2. Craft un Sacrificial Knife
    3. Place-toi au-dessus de l'altar et sacrifie ton sang
    4. Craft un Divination Sigil pour voir ton LP
    5. Upgrade l'altar tier par tier pour débloquer les recettes

    Génération de LP

    • Self-sacrifice early, mob sacrifice later - Sacrificial Knife jusqu'au Well of Suffering
    • Incense Altar double les LP de self-sacrifice (2x efficacité)
    • Les Runes of Self-Sacrifice augmentent le LP gagné

    Mob Sacrifice

    • Dagger of Sacrifice > Sacrificial Knife pour les mobs
    • Utiliser un Well of Suffering avec une mob farm pour LP passive

    Altar Building

    • Les Runes of Speed accélèrent le crafting de l'altar
    • Les Capacity Runes sont nécessaires pour les grosses recettes
    • Ritual Diviner pour placer les rituels automatiquement

    Demon Will

    • Demon Will alimente les Sentient Tools
    • Collecter sur les démons avec la Sentient Sword

    Botania¶

    CurseForge | Guide Enigmatica

    Quick Start

    1. Craft une Lexica Botania (Book + Sapling)
    2. Craft un Petal Apothecary et crée des fleurs
    3. Place un Mana Pool et un Mana Spreader
    4. Crée des Endoflames et alimente-les en charbon
    5. Utilise la Wand of the Forest pour lier les composants

    Addons¶

    • Mythic Botany : Contenu end-game, Alfheim
    • AIOT Botania : Outils combinés

    Génération de Mana

    • [RUSH] Endoflames sont le meilleur départ : peu chers et efficaces
    • Gourmaryllis : meilleure nourriture = plus de mana
    • Munchdew mange les feuilles, bon avec des arbres
    • Agricarnation accélère les crops

    Fleurs Passives

    • Passive flowers decay - Hydroangeas disparaissent avec le temps
    • Préférer les fleurs actives pour le long terme

    Automation

    • Placer Mana Spreader entre les fleurs et le Mana Pool
    • Utiliser Wand of the Forest en Bind Mode pour lier
    • Mana Tablet est ta batterie portable
    • Corporea pour l'autocrafting late-game

    Terrasteel

    • Terrasteel demande de la patience
    • Drop items sur Terrestrial Agglomeration Plate
    • Nécessite beaucoup de mana stocké

    Occultism¶

    CurseForge | Wiki Fandom

    Quick Start

    1. Craft un Dictionary of Spirits (guide in-game)
    2. Utilise la Divination Rod pour trouver les matériaux Otherworld
    3. Mange le fruit des Demon's Dream Seeds pour voir les blocs cachés
    4. Invoque un Crusher Spirit pour doubler tes minerais
    5. Construit un Dimensional Storage Actuator

    Otherworld Materials

    • [RUSH] Divination Rod pour trouver les matériaux (invisibles sinon!)
    • Demon's Dream Seeds - mange le fruit pour voir les minerais cachés
    • Miner dans l'Otherworld Grove pour les ressources rares

    Esprits & Workers

    • Crusher Spirit = ore doubling gratuit early game
    • Transporter Spirits déplacent les items sans pipes
    • Spirit Lamps étendent la zone de travail des esprits
    • Les workers liés ne nécessitent plus de cercles d'invocation

    Pentacles

    • Vérifier les pentacles - mauvais pentacle = mauvais esprit
    • Tu peux utiliser autant de sacrificial bowls que tu veux, pas juste 4!
    • Placer les bowls dans un rayon de 8 blocs du bowl central

    Storage

    • Dimensional Storage Actuator = stockage quasi-illimité
    • Storage Accessor = accès distant (comme un téléphone)
    • Meilleur stockage early-game du jeu!

    Mods Secondaires¶

    EvilCraft

    CurseForge

    Quick Start

    1. Trouve des Dark Gems sous terre
    2. Craft un Blood Infuser et un Sanguinary Pedestal
    3. Collecte du blood en te tenant sur le pedestal
    4. Craft un Broom pour le vol early game!
    • Blood Infuser est le bloc central pour les crafts
    • Sanguinary Pedestal pour collecter le blood des mobs
    • Dark Tank stocke le blood efficacement
    • Environmental Accumulator pour crafts (nécessite météo spécifique)
    • Spirit Furnace pour process les mobs
    • Brooms permettent le vol early game!
    • Exalted Crafter pour auto-crafting alimenté au blood

    Nature's Aura

    CurseForge | Guide Enigmatica

    Quick Start

    1. Craft un Natural Altar (juste bois et pierre!)
    2. Craft un Environmental Eye pour voir l'aura
    3. Génère de l'aura avec le Projectile Generator
    4. Fais pousser un Ancient Oak pour les ressources

    Gestion de l'Aura

    • L'aura se régénère lentement - ne pas sur-consommer
    • Environmental Eye montre les niveaux - essentiel!
    • Effets négatifs quand l'aura est trop basse
    • Natural Altar en premier - station de craft basique
    • Crimson Meal accélère les plantes mieux que bone meal
    • Oak Slabs sous Ancient Oaks attirent les Ancient Saplings
    • Offering Table pour automatiser les recettes
    • Projectile Generator génère de l'aura early

    Iron's Spells 'n Spellbooks

    CurseForge | Wiki

    Quick Start

    1. Explorer les donjons pour trouver des spell scrolls
    2. Craft un Spell Book (leather ou mieux)
    3. Utilise l'Arcane Anvil pour ajouter des spells
    4. Équipe des robes pour boost tes écoles de magie
    • Les spell books ont différentes raretés avec plus de slots
    • Combiner les écoles de magie pour des builds variés
    • Mana regeneration > mana max pour le combat
    • Evasion et Lightning Lance sont très efficaces early
    • Inscription Table pour créer des scrolls

    Spirit

    CurseForge | Guide

    • Système de spawners craftables basé sur les âmes
    • Soul Gem capture les âmes en tuant des mobs
    • Plus d'âmes = tier plus élevé du Soul Cage
    • Les Soul Pedestals autour du cage augmentent le tier
    • Compatible avec Apotheosis pour des spawners très puissants
    • Les cages peuvent être désactivées avec redstone
    Objet Priorité Description
    Soul Cage Spawner craftable
    Soul Gem Capture les âmes
    Crude Soul Gem Version basique (un seul type)
    Soul Pedestal Augmente le tier du cage
    Tier Âmes Effet
    1 16 Spawn basique
    2 64 Spawn plus rapide
    3 128 Spawn encore plus rapide
    4 256 Spawn multiple
    5 512 Spawn maximum

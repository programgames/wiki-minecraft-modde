    Progression Recommandée¶

    Early Game

    • [RUSH] Water Candle de Cyclic (augmente spawn rate en zone)
    • [RUSH] Spikes de Mini Utilities ou Dark Utilities (player kill = XP + rare loot)
    • [RUSH] Mob Catcher de Mob Grinding Utils pour capturer des mobs
    • Cursed Earth pour spawn massif si disponible

    Mid Game

    • Développer les Spawners Apotheosis avec modifiers (Sugar, Clock, Spider Eye)
    • Mob Grinding Utils Mob Masher + Absorption Hopper pour full automation
    • Commencer les Data Models de Hostile Neural Networks

    Late Game

    • Spawners Apotheosis maxés avec Echoing (Nether Star) pour double loot
    • Hostile Neural Networks Self-Aware models pour simulation passive
    • Industrial Foregoing Mob Duplicator + Mob Crusher pour spawning infini

    Synergies entre Mods¶

    Combinaisons Puissantes

    Apotheosis + Mob Grinding Utils

    Apotheosis¶

    CurseForge | Wiki

    Quick Start

    1. Trouve un Spawner vanilla dans un donjon ou mineshaft
    2. Applique du Sugar pour réduire le délai minimum (-5 ticks par sugar)
    3. Applique une Clock pour réduire le délai maximum (-5 ticks par clock)
    4. Applique une Fermented Spider Eye pour +1 mob par spawn
    5. Utilise Cobweb pour No AI (mobs immobiles = farm facile)

    Modifiers de Base

    • Sugar et Clock se stackent - applique-en beaucoup pour spawns très rapides
    • Fermented Spider Eye augmente le nombre de mobs par spawn (+1 par item)
    • Comparator ajoute le contrôle Redstone au spawner
    • Shift+clic droit avec un item vide pour voir les stats du spawner

    Modifiers Avancés

    • Glowstone permet au spawner d'ignorer la lumière
    • Cobweb donne No AI aux mobs (ils ne bougent plus)
    • Combiner No AI avec des Spikes ou Mob Masher pour farm passive

    Attention

    • Les spawners ne peuvent pas être récupérés sans Silk Touch (ou mods)
    • Certains modpacks désactivent le minage de spawners
    • Echoing (Nether Star) coûte très cher mais double le loot

    Configuration Optimale

    Industrial Foregoing¶

    CurseForge | Wiki

    Quick Start

    1. Capture un mob avec le Mob Imprisonment Tool
    2. Place le tool dans un Mob Duplicator avec de l'énergie
    3. Connecte un Mob Crusher à côté de la zone de spawn
    4. Collecte l'Essence (XP liquide) générée par le Crusher
    5. Ajoute des Range Addons pour agrandir les zones de travail

    Mob Crusher

    • Fait 300 dégâts par tick (tue tout sauf Wither en un coup)
    • Ne tue pas les bébés - pratique pour breeding
    • Mode Looting : désactive Essence mais donne loot bonus random
    • Génère de l'Essence (XP liquide) par défaut

    Mob Duplicator

    • Spawne des mobs depuis un Safari Net ou Mob Imprisonment Tool
    • Consomme de l'Essence pour spawner
    • Très efficace combiné avec le Mob Crusher (loop d'Essence)
    • Range Addon pour agrandir la zone de spawn

    Ressources Requises

    • Les machines nécessitent beaucoup d'énergie RF
    • Le Mob Duplicator consomme de l'Essence par spawn
    • Prévoir un bon setup d'énergie avant de commencer

    Hostile Neural Networks¶

    CurseForge | Wiki

    Quick Start

    1. Craft un Data Model vide et place-le dans ton inventaire
    2. Tue des mobs du type voulu - le modèle apprend automatiquement
    3. Le modèle monte en tier : Faulty → Basic → Advanced → Superior → Self-Aware
    4. Place le modèle dans une Simulation Chamber avec de l'énergie
    5. Récupère les Predictions et utilise le Loot Fabricator pour les drops

    Système de Tiers

    • Faulty : Production très lente, beaucoup d'échecs
    • Basic : Production lente
    • Advanced : Production moyenne
    • Superior : Bonne production
    • Self-Aware : Production maximale, aucun échec

    Optimisation

    • Plus le tier est haut, moins la chambre consomme d'énergie
    • Self-Aware = efficacité maximale, rush ce tier
    • Utiliser plusieurs Simulation Chambers en parallèle
    • Les Predictions stackent à 64

    Attention

    • Le modèle doit être dans l'inventaire pour apprendre (pas hotbar seule)
    • Chaque type de mob a besoin de son propre Data Model
    • Les boss peuvent être plus difficiles à modéliser

    Drops Rentables

    Blaze Blaze Rod Brewing, fuel
    Enderman Ender Pearl Téléportation

    Mob Grinding Utils¶

    CurseForge | Wiki

    Quick Start

    1. Craft un Mob Masher et place-le face aux mobs
    2. Enchante le Mob Masher avec Looting et Sharpness
    3. Ajoute un Absorption Hopper pour collecter items ET XP
    4. Utilise des Fans avec upgrades pour pousser les mobs vers le Masher
    5. Place un XP Tap sur l'Absorption Hopper pour extraire l'XP en fluide

    Mob Masher

    • Le Mob Masher est enchantable sur une table d'enchantement!
    • Looting III = plus de drops
    • Sharpness V = plus de dégâts
    • Beheading = plus de têtes de mobs

    Système de Collecte

    • L'Absorption Hopper collecte items ET XP simultanément
    • XP Tap sur le hopper pour extraire l'XP en fluide
    • Tank pour stocker l'XP liquide
    • XP Solidifier pour convertir en bouteilles ou blocs

    Attention

    • Les Fans nécessitent de l'énergie RF
    • L'Ender Inhibitor a un rayon de 8 blocs seulement
    • Les Spikes ont différents tiers - Diamond = player kill

    Setup Anti-Enderman

    • Ender Inhibitor empêche les Endermen de TP (rayon 8)
    • Placer plusieurs inhibitors pour couvrir toute la farm
    • Combiner avec Fans pour les pousser vers le Mob Masher

    Mods Secondaires¶

    Deep Mob Learning

    CurseForge

    Quick Start

    1. Craft un Data Model et place-le dans ton inventaire
    2. Tue des mobs du type correspondant pour l'entraîner
    3. Place le modèle dans une Simulation Chamber
    4. Récupère Living Matter et Pristine Matter
    5. Utilise l'Extraction Chamber ou Loot Fabricator pour les drops

    Système de Tiers

    • Tiers : Faulty → Basic → Advanced → Superior → Self-Aware
    • Plus le tier est haut, plus la production est rapide
    • Pristine Matter est le drop rare et précieux

    Trial System

    • Le Trial Keystone permet de combattre des mobs simulés
    • Récompenses uniques et Glitch Armor materials
    • Glitch Armor est l'armure end-game du mod
    Objet Priorité Description
    Data Model - Apprend d'un type de mob
    Simulation Chamber - Simule les mobs pour drops
    Extraction Chamber - Extrait Living/Pristine Matter
    Loot Fabricator - Convertit la matière en drops
    Trial Keystone Lance des trials pour récompenses
    Glitch Armor Armure end-game puissante

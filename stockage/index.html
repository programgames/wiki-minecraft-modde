    Stockage¶

    • Applied Energistics 2

      Stockage digital avancé avec autocrafting. Le standard end-game pour les modpacks.

      Découvrir

    • Refined Storage

      Alternative plus simple à AE2. Pas de channels, idéal pour débuter.

      Découvrir

    • Functional Storage

      Drawers modulaires avec upgrades. Parfait pour le stockage de masse.

      Découvrir

    • Sophisticated Backpacks

      Sacs à dos tiered avec upgrades. Stockage portable puissant.

      Découvrir

    Progression Recommandée¶

    Early Game

    • [RUSH] Functional Storage pour les drawers de masse
    • [RUSH] Sophisticated Backpacks pour le stockage portable
    • [RUSH] Dank Storage pour le stockage compact
    • [RUSH] Ender Storage pour le partage cross-dimension
    • [RUSH] RFTools Storage Scanner comme mini-AE2
    • Explorer Occultism pour le stockage dimensionnel gratuit

    Mid Game

    • Simple Storage Network pour accès centralisé aux coffres
    • Transition vers AE2 ou Refined Storage
    • Storage Bus sur les drawers existants pour intégration
    • Setup de base pour l'autocrafting

    Late Game

    • AE2 complet avec tous les addons (Mega Cells, Advanced AE)
    • QIO de Mekanism pour accès wireless illimité
    • Autocrafting massif avec CPU multi-core
    • Integration complète avec toutes les machines

    Synergies entre Mods¶

    Combinaisons Puissantes

    AE2 + Functional Storage

    AE2 + Mekanism QIO
    Le meilleur des deux mondes : autocrafting + wireless
    Sophisticated Backpacks + Curios
    Backpack en slot Curio = toujours accessible
    Ender Storage + Pipes
    Transfert cross-dimension automatisé
    RFTools Storage + Modular Routers

    Applied Energistics 2¶

    CurseForge | Guide Officiel

    Quick Start

    1. Mine des Certus Quartz et Meteorites pour les presses
    2. Craft un Charger et charge du Certus en Charged Certus
    3. Crée des Fluix Crystals en jetant Charged Certus + Redstone + Nether Quartz dans l'eau
    4. Craft un Inscriber pour fabriquer les processeurs
    5. Construit ton premier ME Controller + ME Drive + Terminal

    Addons¶

    • Advanced AE : Export bus amélioré, tour à craft
    • Mega Cells : Disques massifs (256k+)
    • AE2 Wireless Terminals : Accès sans fil
    • AE Infinity Booster : Range wireless infinie
    • Applied Mekanistics : Stockage gaz/slurry
    • ME Requester : Auto-stock automatique

    Channels & Cables

    • Dense Smart Cable = 32 channels (vs 8 pour câble normal)
    • P2P Tunnels transportent 32 channels dans un seul câble
    • Quartz Fiber transfère l'énergie sans connecter les channels
    • Les Molecular Assemblers ne consomment pas de channel mais en transfèrent 8

    Autocrafting

    • Pattern Provider entouré de 6 Molecular Assemblers = setup optimal
    • Acceleration Cards dans les Assemblers = plus de vitesse
    • Blocking Mode pour machines qui ne prennent qu'un batch (ex: Energizing Orb)
    • Substitution dans les patterns pour recettes avec variantes

    Attention

    • Crystal Growth Accelerators autour des cristaux = x17 plus rapide
    • Le Controller est obligatoire pour plus de 8 channels
    • Ne pas oublier l'Energy Acceptor pour alimenter le réseau

    Setup Optimal

    • Formation Plane pour placer des blocs automatiquement
    • Level Emitter pour redstone quand un item atteint un seuil
    • Storage Bus sur chest comme buffer temporaire pour les crafts
    Objet Priorité Description
    ME Controller RUSH Cerveau du réseau (obligatoire >8 channels)
    ME Drive Contient jusqu'à 10 disques de stockage
    Storage Cells (1k→256k) Plus gros = plus de types mais même espace
    Crafting Terminal Interface utilisateur avec crafting intégré
    Pattern Provider Envoie les items aux machines externes
    Molecular Assembler Craft les recettes internes
    Import/Export Bus Transfert automatique
    Storage Bus Connecte inventaires externes au réseau
    Inscriber RUSH Fabrique les processeurs (crucial early)
    Charger Charge outils et Certus Quartz
    ME Wireless Access Point Accès sans fil dans la zone

    Refined Storage¶

    CurseForge | Guide Officiel

    Quick Start

    1. Craft un Controller (cœur du réseau)
    2. Craft un Disk Drive et des Storage Disks
    3. Connecte avec des Cables et ajoute un Grid
    4. Craft un Crafting Grid pour le crafting intégré
    5. Ajoute des Crafters + Patterns pour l'autocrafting

    Addons¶

    • Extra Disks : Disques plus gros
    • Cable Tiers : Câbles plus rapides
    • Extra Storage : Plus de stockage

    Différences avec AE2¶

    Aspect Refined Storage AE2
    Channels Non Oui (8/32)
    Complexité Simple Complexe
    Autocrafting Direct Plus optimisable

    Configuration

    • Pas de système de channels = plus simple à configurer
    • Le Controller est le cœur (un seul par réseau)
    • Les câbles colorés permettent de séparer les réseaux
    • Wireless Transmitter pour accès sans fil

    Autocrafting

    • Crafter + Pattern pour l'autocrafting
    • Plusieurs Crafters peuvent être chaînés
    • Processing patterns pour machines externes

    Setup Rapide

    • External Storage connecte des inventaires externes
    • Importer/Exporter pour transfert automatique
    • Detector émet redstone selon le stock
    Objet Priorité Description
    Controller RUSH Cœur du réseau
    Disk Drive Contient les disques de stockage
    Storage Disks (1k→64k) Plus de variantes avec addons
    Grid Interface utilisateur basique
    Crafting Grid Grid avec crafting intégré
    Pattern Grid Crée les patterns d'autocrafting
    Crafter Machine d'autocrafting
    External Storage Connecte inventaires externes
    Importer/Exporter Transfert automatique
    Wireless Transmitter Accès sans fil
    Wireless Grid Accès portable au réseau

    Functional Storage¶

    CurseForge

    Quick Start

    1. Craft des Drawers (1x1, 2x2, ou 4x4)
    2. Place un Storage Controller près des drawers
    3. Utilise le Linking Tool : Left-click controller, Right-click drawer
    4. Ajoute des Upgrades pour augmenter la capacité
    5. Utilise un Compacting Drawer pour ingot/block/nugget automatique

    Controller & Linking

    • Linking Tool : Left-click controller → Right-click drawer pour lier
    • Range par défaut : 8 blocs du controller (upgradable)
    • Le controller donne accès centralisé à tous les drawers liés

    Upgrades Puissants

    • Les upgrades donnent des boosts plus importants que Storage Drawers
    • Ender Drawers synchronisés cross-dimension
    • Void Upgrade détruit le surplus (évite overflow)

    Capacité

    • Plus de slots (2x2, 4x4) = moins de capacité par slot
    • 1x1 Drawer est meilleur pour les items très nombreux

    Automation

    • Collector Upgrade : Vacuum items par terre
    • Puller/Pusher Upgrade : Transfert automatique vers/depuis inventaires
    • Compatible avec AE2 Storage Bus pour intégration
    Objet Priorité Description
    1x1 Drawer RUSH Un slot, capacité maximale par item
    2x2 / 4x4 Drawer Plus de slots, moins de capacité chacun
    Compacting Drawer Stocke ingot + block + nugget auto
    Storage Controller Accès centralisé à tous les drawers liés
    Linking Tool RUSH Lie les drawers au controller
    Ender Drawer Synchronisé cross-dimension
    Void Upgrade Détruit le surplus
    Collector Upgrade Vacuum items par terre
    Puller/Pusher Upgrade Transfert automatique
    Storage Upgrade (tiers) Augmente la capacité

    Sophisticated Backpacks¶

    CurseForge

    Quick Start

    1. Craft un Leather Backpack basique
    2. Upgrade vers Iron → Gold → Diamond → Netherite
    3. Ajoute des Upgrade Slots pour plus de fonctionnalités
    4. Installe des Pickup/Filter Upgrades pour collecte automatique
    5. Utilise le Feeding Upgrade pour manger automatiquement

    Upgrades

    • Les backpacks peuvent être tiered up sans perdre le contenu
    • Chaque tier ajoute des slots de stockage ET d'upgrade
    • Peut être ouvert depuis l'inventaire sans le placer

    Upgrades Essentiels

    • Feeding Upgrade : Mange automatiquement depuis le backpack
    • Magnet Upgrade : Attire les items
    • Crafting Upgrade : Table de craft portable

    Curios Integration

    • Équiper en slot Curio pour accès permanent
    • Keybind pour ouvrir directement

    Automation

    • Pickup Upgrade : Collecte certains items directement
    • Filter Upgrade : Filtre ce qui entre
    • Void Upgrade : Détruit les items spécifiés
    Objet Priorité Description
    Leather Backpack RUSH Premier backpack craftable
    Iron/Gold/Diamond/Netherite Tiers supérieurs
    Feeding Upgrade Auto-feed depuis le backpack
    Pickup Upgrade Collecte items au sol
    Filter Upgrade Filtre les entrées
    Crafting Upgrade Table de craft portable
    Magnet Upgrade Attire les items
    Void Upgrade Détruit items spécifiés
    Stack Upgrade Augmente la taille des stacks
    Tool Swapper Upgrade Swap auto d'outils

    Mods Secondaires¶

    Storage Drawers

    CurseForge

    Quick Start

    1. Craft des Drawers (1x1, 1x2, 2x2) avec du bois
    2. Place-les et right-click avec des items pour stocker
    3. Craft un Drawer Controller pour accès centralisé
    4. Connecte les drawers au controller avec des Trim ou adjacence
    5. Ajoute des Upgrades pour augmenter la capacité

    Types de Drawers

    • 1x1 : Maximum de capacité pour un seul type
    • 1x2 : Deux types, capacité divisée
    • 2x2 : Quatre types, capacité divisée par 4
    • Compacting Drawer : Auto-conversion ingot/block/nugget

    Upgrades

    • Storage Upgrade (I → V) : Augmente la capacité
    • Void Upgrade : Détruit le surplus
    • Redstone Upgrade : Signal selon le remplissage

    Le mod original

    • Functional Storage est un fork moderne
    • Storage Drawers est le classique qui a inspiré beaucoup
    • Compatible avec la plupart des mods de transport
    Objet Priorité Description
    Basic Drawer (1x1) RUSH Meilleure capacité par type
    Compacting Drawer Auto-conversion ingot/block/nugget
    Drawer Controller Accès centralisé à tous les drawers
    Drawer Key Configure le lock/unlock
    Storage Upgrades Augmente la capacité (Tier I-V)
    Void Upgrade Détruit le surplus automatiquement
    Iron Chests

    CurseForge

    Quick Start

    1. Craft un Iron Chest (coffre + lingots de fer)
    2. Upgrade vers Gold → Diamond → Obsidian → Crystal
    3. Utilise les Chest Upgrades pour upgrade sans vider
    4. Le Crystal Chest est transparent et montre le contenu
    5. Compatible avec hoppers et tous les mods de transport

    Tiers de Coffres

    Tier Slots Notes
    Iron 54 Double d'un coffre
    Gold 81 Bon rapport
    Diamond 108 Large capacité
    Obsidian 108 Résiste aux explosions
    Crystal 108 Transparent!

    Upgrades In-Place

    • Les Chest Upgrades permettent d'upgrade sans vider
    • Iron → Gold, Gold → Diamond, etc.
    • Très pratique pour ne pas tout déplacer
    Objet Priorité Description
    Iron Chest RUSH 54 slots, première upgrade
    Diamond Chest 108 slots
    Crystal Chest 108 slots, transparent
    Chest Upgrades Upgrade sans vider
    Colossal Chests

    CurseForge

    Quick Start

    1. Craft des Colossal Chest Wall blocks
    2. Construit une structure cube de n'importe quelle taille (2x2x2 minimum)
    3. Place un Colossal Chest Core quelque part dans la structure
    4. Right-click pour former le coffre géant
    5. Plus c'est grand, plus il y a de stockage!

    Tailles et Capacité

    • Taille minimum : 2x2x2 (intérieur)
    • Taille maximum : configurable (souvent 20x20x20)
    • La capacité scale avec le volume
    • Un 5x5x5 = des centaines de slots

    Matériaux

    • Wood : Basique, économique
    • Copper : Plus de capacité
    • Iron : Encore plus
    • Diamond : Maximum de capacité par bloc

    Performance

    • Les très grands coffres peuvent être lents à ouvrir
    • Utilise plutôt plusieurs petits que un énorme
    Objet Priorité Description
    Colossal Chest Wall Blocs pour construire la structure
    Colossal Chest Core Active la structure
    Interface Connexion pour pipes/AE2
    Simple Storage Network

    CurseForge

    Quick Start

    1. Craft un Storage Network Master (cerveau)
    2. Craft des Network Cables pour connecter
    3. Craft des Storage Link Cables pour lier les coffres
    4. Place un Storage Request Table pour accéder au réseau
    5. Tous les coffres connectés sont maintenant accessibles!

    Système Simple

    • Alternative très simple à AE2/RS
    • Pas de channels, pas de complexité
    • Parfait pour débuter l'automatisation

    Composants

    • Master : Un seul par réseau
    • Link Cable : Connecte un inventaire
    • Import/Export Cable : Automation basique
    • Request Table : Interface utilisateur

    Limitations

    • Pas d'autocrafting intégré
    • Pas de stockage digital (utilise les coffres existants)
    • Pour les systèmes plus complexes, passer à AE2/RS
    Objet Priorité Description
    Storage Network Master RUSH Cerveau du réseau
    Network Cable Connecte les éléments
    Storage Link Cable Lie un coffre au réseau
    Storage Request Table Interface utilisateur
    Import/Export Cable Automation basique
    Dank Storage

    CurseForge

    Quick Start

    1. Craft un Dank 1 basique
    2. Upgrade vers des tiers supérieurs pour plus de capacité
    3. Configure les slots pour les items à stocker
    4. Utilise le mode Void pour les items indésirables
    • Stockage très compact de plusieurs stacks d'un même item
    • Plusieurs tiers avec capacité croissante
    • Peut servir de trash pour les items indésirables
    • Portable et plaçable
    Objet Priorité Description
    Dank 1-7 Tiers croissants de stockage
    Dock Place le Dank comme bloc
    RFTools Storage

    CurseForge | Wiki GitHub

    Quick Start

    1. Craft un Storage Scanner
    2. Ajoute des Modular Storage ou coffres normaux autour
    3. Craft un Storage Tablet pour accès portable
    4. Configure les filtres de recherche

    Mini-AE2

    • [RUSH] Storage Scanner + Modular Storage + Tablet = système de stockage early sans AE2
    • Le Scanner détecte automatiquement les changements d'inventaire
    • Beaucoup moins cher qu'AE2 ou Refined Storage
    • Screens pour afficher le contenu des storages
    • Compatible avec tous les inventaires vanilla et moddés
    Objet Priorité Description
    Storage Scanner RUSH Recherche dans tous les inventaires
    Modular Storage Stockage modulaire
    Storage Tablet Accès portable au Scanner
    Screen Affiche les informations
    Occultism Storage

    CurseForge | Wiki Fandom

    Quick Start

    1. Suivre la progression Occultism (Dictionary of Spirits)
    2. Invoque un Dimensional Storage Actuator
    3. Craft un Storage Accessor pour accès portable
    4. Ajoute des Storage Stabilizers pour plus de capacité

    Stockage Dimensionnel

    • Dimensional Storage Actuator = stockage quasi-illimité
    • Storage Accessor = accès portable (comme un téléphone)
    • Le système fonctionne cross-dimension
    • Pas besoin de câbles = très propre
    • Utilise des esprits pour transporter les items
    • Storage Stabilizers augmentent la capacité
    • Les esprits peuvent auto-craft avec les bonnes configurations
    Objet Priorité Description
    Dimensional Storage Actuator RUSH Bloc de stockage principal
    Storage Accessor Accès portable au stockage
    Storage Stabilizer Tier 1-4 Augmente la capacité
    Stable Wormhole Accès distant fixe
    Storage Remote Accès wireless
    Transporter Spirits Déplacent items automatiquement
    Mekanism QIO

    CurseForge

    Quick Start

    1. Craft un QIO Drive Array
    2. Craft des QIO Drives et insère-les dans l'array
    3. Craft un QIO Dashboard pour l'interface
    4. Crée une Frequency pour lier les composants
    5. Craft un Portable QIO Dashboard pour accès wireless

    End-Game Wireless

    • Système de stockage end-game de Mekanism
    • Accès wireless sans limite de range (même cross-dimension)
    • Frequency partagée = même stockage partout

    Coût

    • Très efficace mais coûteux en ressources
    • Nécessite progression Mekanism avancée
    • Compatible avec l'autocrafting via QIO Crafting Window
    • Intégration native avec les machines Mekanism
    Objet Priorité Description
    QIO Drive Array Contient les QIO Drives
    QIO Dashboard Interface utilisateur
    QIO Drives (tiers) Stockage (plusieurs capacités)
    QIO Importer/Exporter Transfert automatique
    Portable QIO Dashboard RUSH Accès portable wireless
    QIO Crafting Window Autocrafting intégré
    QIO Redstone Adapter Contrôle redstone
    Ender Storage

    CurseForge

    Quick Start

    1. Craft un Ender Chest avec des couleurs de laine
    2. Les coffres avec mêmes couleurs partagent le contenu
    3. Place un Diamond sur le loquet pour rendre le coffre privé
    4. Craft un Ender Pouch pour accès portable

    Code Couleur

    • 3 couleurs configurables par coffre
    • Synchronisés cross-dimension instantanément
    • Parfait pour transfert entre bases

    Privé vs Public

    • Diamond sur le loquet = coffre privé (lié au joueur)
    • Sans diamond = accessible par tous avec le même code
    • Ender Tanks pour les fluides (même système de couleurs)
    • Automatisable avec hoppers/pipes
    Objet Priorité Description
    Ender Chest RUSH Coffre partagé par couleurs
    Ender Tank Tank partagé par couleurs
    Ender Pouch Accès portable à un Ender Chest

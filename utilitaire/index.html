    Apotheosis¶

    CurseForge | All The Guides

    Quick Start

    1. Craft un Enchanting Table et entoure-la de Bookshelves
    2. Upgrade vers des Hellshelves pour atteindre niveau 60+
    3. Craft une Extraction Table pour retirer les enchants sans détruire
    4. Trouve des Gems en tuant des mobs pour ajouter des affixes
    5. Utilise la Gem Cutting Table pour upgrade tes gems

    Addons¶

    • Apothic Attributes
    • Apothic Spawners
    • Apothic Enchanting

    Système d'Enchantement

    • Le niveau max est 150 sur la table d'enchantement (au lieu de 30 vanilla)
    • Eterna = puissance d'enchantement (comme les niveaux vanilla)
    • Quanta = variance/randomness des enchants
    • Arcana = chance d'obtenir des enchants rares

    Types de Bookshelves

    • Bookshelf : +2 Eterna (vanilla)
    • Hellshelf : +4 Eterna (double puissance)
    • Seashelf : Augmente chance treasure enchants
    • Enderscribe Shelf : Avec un livre d'enchant = bias vers cet enchant

    Spawners

    • Right-click avec des items pour modifier les spawners
    • Sugar : -5 ticks délai minimum
    • Clock : -5 ticks délai maximum
    • Fermented Spider Eye : +1 mob par spawn
    • Redstone Control possible pour automation

    Gems & Affixes

    • Les gems s'insèrent dans l'équipement pour des bonus permanents
    • Les affixes Ancient (texte doré) sont les plus puissants
    • Farm les boss mobs pour du loot avec affixes rares
    Objet Priorité Description
    Hellshelf Bookshelf x2 puissance (+4 Eterna)
    Seashelf Augmente chance treasure enchants
    Infused Hellshelf Version améliorée du Hellshelf
    Enderscribe Shelf Bias vers un enchant spécifique
    Extraction Table RUSH Retire enchants sans détruire l'item
    Reforging Table Reroll les affixes de l'équipement
    Salvaging Table Récupère les enchants des items
    Gem Cutting Table Upgrade et coupe les gems
    Ender Library Stocke les livres enchantés
    Ender Lead Lead qui fonctionne sur Endermen
    Spawner Modifiers Items pour améliorer spawners

    Modular Routers¶

    CurseForge | Wiki GitHub

    Quick Start

    1. Craft un Modular Router (iron + chest + redstone)
    2. Craft un Puller Module pour tirer des items d'un inventaire
    3. Craft un Sender Module pour envoyer vers un autre inventaire
    4. Ajoute des Speed Upgrades pour accélérer le router
    5. Ajoute un Stack Upgrade - sans lui, 1 item à la fois!

    Performance

    • Un router peut remplacer 20+ hoppers avec de meilleures performances
    • Les Speed Upgrades sont multiplicatifs - stack pour vitesse folle
    • Les modules s'exécutent de gauche à droite dans l'interface

    Modules Essentiels

    • Sender Mk2 peut envoyer cross-dimension!
    • Vacuum Module collecte les items par terre (6 blocs, upgradable)
    • Inspection Module filtre par NBT, dégâts, enchants
    • Detector Module émet redstone selon conditions

    Attention

    • Stack Upgrade est ESSENTIEL - sans lui, seulement 1 item/tick
    • Penser à configurer les filtres pour éviter les boucles

    Automation Crops

    • Placer router au-dessus d'un coffre de récolte
    • Puller Module vers le coffre
    • Sender Module vers le stockage
    • Ajouter Speed + Stack Upgrades = farm automatisée
    Objet Priorité Description
    Modular Router RUSH Bloc central du mod
    Puller Module Tire items d'un inventaire adjacent
    Sender Module Envoie items vers inventaire distant
    Sender Mk2 Envoie cross-dimension
    Vacuum Module Collecte items par terre
    Dropper Module Drop items dans le monde
    Placer Module Place des blocs
    Breaker Module Casse des blocs
    Activator Module Utilise items comme un joueur
    Speed Upgrade RUSH Accélère le router
    Stack Upgrade RUSH Traite plus d'items par tick
    Range Upgrade Augmente la portée
    Camouflage Upgrade Cache le router (déco)

    Functional Storage¶

    CurseForge

    Quick Start

    1. Craft un 1x1 Drawer pour stocker un type d'item en masse
    2. Craft un Storage Controller pour accès centralisé
    3. Craft un Linking Tool et left-click sur le controller
    4. Right-click sur chaque drawer pour les lier (même à distance!)
    5. Ajoute des Storage Upgrades pour augmenter la capacité

    Linking System

    • Linking Tool : Left-click sur controller pour copier, right-click sur drawer pour lier
    • Les drawers peuvent être liés sans contact direct avec le controller (range 8)
    • Ender Drawers peuvent être synchronisés cross-dimension

    Types de Drawers

    • 1x1 Drawer : 1 type d'item, capacité maximale
    • 2x2 Drawer : 4 types, capacité divisée
    • 4x4 Drawer : 16 types, moins de capacité chacun
    • Compacting Drawer : Stocke item + versions compressées auto

    Upgrades

    • Les Storage Upgrades sont plus puissants que dans d'autres mods
    • Penser à upgrader les drawers pour items en masse (cobble, etc.)

    Setup Recommandé

    • 1x1 pour les items en grande quantité
    • 2x2 pour les items moyens
    • Compacting pour les métaux (nugget/ingot/block auto)

    Cyclic¶

    CurseForge

    Quick Start

    1. Craft un Ender Book pour sauvegarder des positions de téléportation
    2. Craft un Harvester et place-le devant tes crops
    3. Craft un Sprinkler au-dessus des crops pour accélérer la croissance
    4. Craft un Collector pour vacuum les items récoltés
    5. Setup le Wireless Energy pour alimenter sans câbles

    Transport & Téléportation

    • [RUSH] Ender Book = waystone portable (sauvegarde positions)
    • Ender Pouch + Ender Chest = transport d'items wireless cross-dimension
    • Player Launcher excellent pour le transport early game

    Farming

    • [RUSH] Sprinkler accélère croissance des crops
    • [RUSH] Water Candle augmente spawn mobs (mob farms)
    • Harvester auto-récolte ET replante automatiquement
    • Terra Preta = sol qui accélère les crops

    Wireless Systems

    • Wireless Energy/Item/Fluid : pas de câbles, juste transmitter + receiver
    • Fonctionne cross-dimension!

    Utilitaires Pratiques

    • Sleeping Mat : skip la nuit sans changer le spawn
    • Glove of Building : place blocs en 3x3
    • Heart Containers : +HP permanent stackable
    • Monster Ball : capture des mobs

    Mods Secondaires¶

    Tinker's Construct

    CurseForge | Material Stats Wiki

    Quick Start

    1. Construit une Smeltery (seared bricks en carré creux)
    2. Craft une Tool Station pour créer tes outils
    3. Utilise Paper pour les bindings (+1 modifier gratuit)
    4. Ajoute Moss pour réparation automatique
    5. Upgrade avec Redstone (haste) et Lapis (luck)

    Réparation

    • Réparer est GRATUIT avec les matériaux (pas de coût XP!)
    • Utiliser des nuggets pour réparer = économie de ressources
    • Moss modifier = réparation automatique avec le temps

    Meilleurs Matériaux

    • Manyullyn : Plus hauts dégâts
    • Cobalt : Mining le plus rapide
    • Paper : +1 modifier slot (binding/handle)

    Smeltery

    • Smeltery double les minerais - toujours process avant de smelt!
    • Rotten Flesh peut être fondu en blood

    Builds Recommandés

    • Cleaver : meilleure arme early (gros dégâts + beheading)
    • Mattock > Pickaxe early : mine pierre ET bois
    • Rapier : ignore armure, attaque rapide

    Seared Bricks Blocs de base smeltery
    Sledgehammer RUSH Mine vite sable/gravel/cobble
    Cleaver Gros dégâts + beheading
    Rapier Ignore armure
    Slimesling Propulsion rapide

    Waystones

    CurseForge

    Quick Start

    1. Trouve une Waystone dans un village ou structure
    2. Interagis avec pour l'activer et la nommer
    3. Craft un Warp Stone pour téléportation portable
    4. Place tes propres Waystones à ta base et spots importants

    Téléportation

    • [RUSH] Activer chaque Waystone trouvée!
    • Warp Stone = téléportation vers n'importe quelle Waystone activée
    • Les Waystones peuvent être craftées et placées
